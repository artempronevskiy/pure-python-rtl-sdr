–°–æ–µ–¥–∏–Ω–∏–≤ –∏—Ö –ø–∞—Ä–∞–ª–µ–ª–ª—å–Ω–æ, –ø–æ–ª—É—á–∞—é—Ç **LC-–∫–æ–Ω—Ç—É—Ä**. –ù–∞ –æ–¥–Ω–æ–π —Å—Ç—Ä–æ–≥–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–π —á–∞—Å—Ç–æ—Ç–µ `f_res = 1 / (2*œÄ * sqrt(L*C))` (–§–æ—Ä–º—É–ª–∞ –¢–æ–º—Å–æ–Ω–∞) —Å–æ–ø—Ä–æ—Ç–∏–≤–ª–µ–Ω–∏—è —É—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è (`X_c = X_l`). –ö–æ–Ω—Ç—É—Ä –≤—Ö–æ–¥–∏—Ç –≤ —Ä–µ–∑–æ–Ω–∞–Ω—Å, —É—Å–∏–ª–∏–≤–∞—è –Ω—É–∂–Ω—É—é —á–∞—Å—Ç–æ—Ç—É –≤ —Ç—ã—Å—è—á–∏ —Ä–∞–∑ –∏ –±–ª–æ–∫–∏—Ä—É—è –≤–µ—Å—å –æ—Å—Ç–∞–ª—å–Ω–æ–π —ç—Ñ–∏—Ä. –¢–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç –∞–Ω–∞–ª–æ–≥–æ–≤–∞—è "–∫—Ä—É—Ç–∏–ª–∫–∞ –Ω–∞ –º–∞–≥–Ω–∏—Ç–æ–ª–µ".

## 3. –ì–µ—Ç–µ—Ä–æ–¥–∏–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ (–û–±—Ä–∞–±–æ—Ç–∫–∞ —Å–ø–µ–∫—Ç—Ä–∞ –∞–Ω–∞–ª–æ–≥–æ–≤–æ–π –º–∞—Ç–µ–º–∞—Ç–∏–∫–æ–π)

–ß—Ç–æ–±—ã –ø—Ä–∏–Ω—è—Ç—å —Å—Ç–∞–Ω—Ü–∏—é –Ω–∞ 100 –ú–ì—Ü, –Ω–∞–º –Ω—É–∂–µ–Ω –∏–¥–µ–∞–ª—å–Ω—ã–π —Ñ–∏–ª—å—Ç—Ä –Ω–∞ 100 –ú–ì—Ü. –ß—Ç–æ–±—ã –ø—Ä–∏–Ω—è—Ç—å —Å—Ç–∞–Ω—Ü–∏—é –Ω–∞ 105 –ú–ì—Ü, –Ω—É–∂–µ–Ω —Ñ–∏–ª—å—Ç—Ä –Ω–∞ 105 –ú–ì—Ü. –°–æ–∑–¥–∞—Ç—å –∫–∞—á–µ—Å—Ç–≤–µ–Ω–Ω—ã–π –ø–µ—Ä–µ—Å—Ç—Ä–∞–∏–≤–∞–µ–º—ã–π —Ñ–∏–ª—å—Ç—Ä —Ñ–∏–∑–∏—á–µ—Å–∫–∏ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ ‚Äî –µ–≥–æ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏ –±—É–¥—É—Ç "–ø–ª—ã—Ç—å".

–ò–Ω–∂–µ–Ω–µ—Ä—ã –ø—Ä–∏–¥—É–º–∞–ª–∏ –≥–µ–Ω–∏–∞–ª—å–Ω—ã–π –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–π —Ö–∞–∫ ‚Äî **–°—É–ø–µ—Ä–≥–µ—Ç–µ—Ä–æ–¥–∏–Ω**.
–í–º–µ—Å—Ç–æ —Ç–æ–≥–æ —á—Ç–æ–±—ã –¥–≤–∏–≥–∞—Ç—å —Å–≤–æ–∏ —Ñ–∏–ª—å—Ç—Ä—ã –∑–∞ —Å–∏–≥–Ω–∞–ª–æ–º, –º—ã –±—É–¥–µ–º **—Å–¥–≤–∏–≥–∞—Ç—å –≤–µ—Å—å —Ä–∞–¥–∏–æ—ç—Ñ–∏—Ä –≤–Ω–∏–∑**, –∫ –Ω–∞—à–µ–º—É –æ–¥–Ω–æ–º—É —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–º—É –∏–¥–µ–∞–ª—å–Ω–æ–º—É —Ñ–∏–ª—å—Ç—Ä—É!

### –§–∏–∑–∏–∫–∞ –∏ –º–∞—Ç–µ–º–∞—Ç–∏–∫–∞ –Ω–µ–ª–∏–Ω–µ–π–Ω–æ–≥–æ —Å–º–µ—Å–∏—Ç–µ–ª—è
–ß—Ç–æ–±—ã —Å–¥–≤–∏–Ω—É—Ç—å —Å–ø–µ–∫—Ç—Ä, –º–∞—Ç–µ–º–∞—Ç–∏–∫–∞ —Ç—Ä–µ–±—É–µ—Ç **—É–º–Ω–æ–∂–∏—Ç—å** —Å–∏–≥–Ω–∞–ª —Å –∞–Ω—Ç–µ–Ω–Ω—ã –Ω–∞ —Å–∏–≥–Ω–∞–ª –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–≥–æ –≤ –ø—Ä–∏–µ–º–Ω–∏–∫ –ª–æ–∫–∞–ª—å–Ω–æ–≥–æ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–∞ (–≥–µ—Ç–µ—Ä–æ–¥–∏–Ω–∞, LO). –í —Ñ–∏–∑–∏–∫–µ —É–º–Ω–æ–∂–µ–Ω–∏–µ —Ä–µ–∞–ª–∏–∑—É–µ—Ç—Å—è –ø—Ä–æ–ø—É—Å–∫–∞–Ω–∏–µ–º –¥–≤—É—Ö —Ç–æ–∫–æ–≤ —á–µ—Ä–µ–∑ –Ω–µ–ª–∏–Ω–µ–π–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç (–Ω–∞–ø—Ä–∏–º–µ—Ä, –¥–∏–æ–¥ –∏–ª–∏ —Ç—Ä–∞–Ω–∑–∏—Å—Ç–æ—Ä –Ω–∞ –∏–∑–≥–∏–±–µ –í–ê–•).

–ü—É—Å—Ç—å —Å—Ç–∞–Ω—Ü–∏—è –≤–µ—â–∞–µ—Ç –Ω–∞ —á–∞—Å—Ç–æ—Ç–µ `F_rf = 100 –ú–ì—Ü`: `U_rf = cos(2*œÄ * 100 * t)`
–ú—ã –≥–µ–Ω–µ—Ä–∏—Ä—É–µ–º –ª–æ–∫–∞–ª—å–Ω—ã–π —Å–∏–≥–Ω–∞–ª `F_lo = 100 –ú–ì—Ü`: `U_lo = cos(2*œÄ * 100 * t)`

–§–∏–∑–∏—á–µ—Å–∫–∏ –ø–µ—Ä–µ–º–Ω–æ–∂–∞–µ–º –∏—Ö –≤ —Å–º–µ—Å–∏—Ç–µ–ª–µ:
`U_mix = cos(2*œÄ * 100 * t) * cos(2*œÄ * 100 * t)`

–ü—Ä–∏–º–µ–Ω—è–µ–º –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–æ–µ —Ç—Ä–∏–≥–æ–Ω–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–æ–µ —Ç–æ–∂–¥–µ—Å—Ç–≤–æ `cos(a)*cos(b) = 0.5*[cos(a-b) + cos(a+b)]`:
`U_mix = 0.5 * cos(2*œÄ * [100 - 100]*t)  +  0.5 * cos(2*œÄ * [100 + 100]*t)`

**–†–µ–∑—É–ª—å—Ç–∞—Ç:** –ß–∞—Å—Ç–æ—Ç–∞ `100 –ú–ì—Ü` –∏—Å—á–µ–∑–ª–∞! –í–º–µ—Å—Ç–æ –Ω–µ–µ —Ä–æ–¥–∏–ª–∏—Å—å –¥–≤–µ –Ω–æ–≤—ã–µ:
1. **–†–∞–∑–Ω–æ—Å—Ç–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞:** `100 - 100 = 0 –ú–ì—Ü`. (–ë–∞–∑–æ–≤–∞—è –ø–æ–ª–æ—Å–∞, Baseband). –°–∏–≥–Ω–∞–ª –ø–µ—Ä–µ–Ω–µ—Å–µ–Ω –≤–Ω–∏–∑!
2. **–°—É–º–º–∞—Ä–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞:** `100 + 100 = 200 –ú–ì—Ü`. –≠—Ç–æ—Ç –≤—ã—Å–æ–∫–æ—á–∞—Å—Ç–æ—Ç–Ω—ã–π –º—É—Å–æ—Ä –º–≥–Ω–æ–≤–µ–Ω–Ω–æ –≥–∞—Å–∏—Ç—Å—è –ø—Ä–æ—Å—Ç–µ–π—à–∏–º –∫–æ–Ω–¥–µ–Ω—Å–∞—Ç–æ—Ä–æ–º (–§–ù–ß).

```python
# –§–∏–∑–∏–∫–∞ —Å–º–µ—Å–∏—Ç–µ–ª—è –≤ Python
import math

def analog_mixer(rf_signal_volts: list[float], f_lo: float, sample_rate: float) -> list[float]:
    mixed = []
    for i, volts in enumerate(rf_signal_volts):
        time = i / sample_rate
        # –ì–µ–Ω–µ—Ä–∞—Ü–∏—è –æ–ø–æ—Ä–Ω–æ–≥–æ —Å–∏–≥–Ω–∞–ª–∞ –≥–µ—Ç–µ—Ä–æ–¥–∏–Ω–∞
        oscillator = math.cos(2.0 * math.pi * f_lo * time)
        # –§–∏–∑–∏—á–µ—Å–∫–æ–µ –ø–µ—Ä–µ–º–Ω–æ–∂–µ–Ω–∏–µ –Ω–∞–ø—Ä—è–∂–µ–Ω–∏–π (—Å–æ–∑–¥–∞–µ—Ç f_rf - f_lo –∏ f_rf + f_lo)
        mixed.append(volts * oscillator)
    # –î–∞–ª–µ–µ —Ç—Ä–µ–±—É–µ—Ç—Å—è —Ü–∏—Ñ—Ä–æ–≤–æ–π –§–ù–ß (Low-Pass Filter)
    return mixed
```

   - **–ê–Ω–∞–ª–æ–≥–æ–≤—ã–π –§–ù–ß:** –û—Ç—Ä–µ–∑–∞–µ—Ç –ª–∏—à–Ω–∏–π –º—É—Å–æ—Ä –∏ –∑–µ—Ä–∫–∞–ª—å–Ω—ã–µ —á–∞—Å—Ç–æ—Ç—ã –∞–ø–ø–∞—Ä–∞—Ç–Ω–æ.

2. **–ú–∏–∫—Ä–æ—Å—Ö–µ–º–∞ –ê–¶–ü –∏ DDC (RTL2832U):** –ú–æ—Å—Ç –≤ –∫–æ–º–ø—å—é—Ç–µ—Ä.

   - **DDC (Digital Down-Converter):** –ö—Ä–µ–º–Ω–∏–µ–≤—ã–π —Ü–∏—Ñ—Ä–æ–≤–æ–π —Å–æ–ø—Ä–æ—Ü–µ—Å—Å–æ—Ä, –∫–æ—Ç–æ—Ä—ã–π –ø–µ—Ä–µ–Ω–æ—Å–∏—Ç –æ—Å—Ç–∞–≤—à—É—é—Å—è IF —á–∞—Å—Ç–æ—Ç—É —Ä–æ–≤–Ω–æ –≤ `0 –ì—Ü`, –≤—ã–ø–æ–ª–Ω—è–µ—Ç –ø–µ—Ä–≤—É—é –≥—Ä—É–±—É—é —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏—é –∏ –ø—Ä–æ—Ä–µ–∂–∏–≤–∞–µ—Ç (–¥–µ—Ü–∏–º–∏—Ä—É–µ—Ç) –ø–æ—Ç–æ–∫ –¥–∞–Ω–Ω—ã—Ö –¥–æ —Å–∫–æ—Ä–æ—Å—Ç–∏ USB 2.0 (–¥–æ 3.2 –ú–µ–≥–∞–±–∞–π—Ç/—Å).

## 5. –¢–µ–æ—Ä–µ–º–∞ –ö–æ—Ç–µ–ª—å–Ω–∏–∫–æ–≤–∞ (–ù–∞–π–∫–≤–∏—Å—Ç–∞) –∏ –ê–¶–ü

–ê–¶–ü –ø–µ—Ä–µ–≤–æ–¥–∏—Ç –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–µ —Ñ–∏–∑–∏—á–µ—Å–∫–æ–µ –≤—Ä–µ–º—è `t` –≤ –¥–∏—Å–∫—Ä–µ—Ç–Ω—ã–π –º–∞—Å—Å–∏–≤ –∏–Ω–¥–µ–∫—Å–æ–≤ `n`.

**–§—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω—ã–π –ø—Ä–µ–¥–µ–ª —Ñ–∏–∑–∏–∫–∏ (–¢–µ–æ—Ä–µ–º–∞ –ù–∞–π–∫–≤–∏—Å—Ç–∞-–®–µ–Ω–Ω–æ–Ω–∞-–ö–æ—Ç–µ–ª—å–Ω–∏–∫–æ–≤–∞):** 
> –ß—Ç–æ–±—ã –¥–∏—Å–∫—Ä–µ—Ç–Ω—ã–π –º–∞—Å—Å–∏–≤ —á–∏—Å–µ–ª –º–æ–≥ —Å–æ–¥–µ—Ä–∂–∞—Ç—å –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ —Å–∏–Ω—É—Å–æ–∏–¥–µ —á–∞—Å—Ç–æ—Ç–æ–π `F_max` –±–µ–∑ –ø–æ—Ç–µ—Ä—å, –ê–¶–ü –æ–±—è–∑–∞–Ω "—Ñ–æ—Ç–æ–≥—Ä–∞—Ñ–∏—Ä–æ–≤–∞—Ç—å" –Ω–∞–ø—Ä—è–∂–µ–Ω–∏–µ —Å–æ —Å–∫–æ—Ä–æ—Å—Ç—å—é `F_s > 2 * F_max`. 

–ù–∞ –∫–∞–∂–¥—É—é –≤–æ–ª–Ω—É –¥–æ–ª–∂–Ω–æ –ø—Ä–∏—Ö–æ–¥–∏—Ç—å—Å—è –º–∏–Ω–∏–º—É–º –¥–≤–∞ –∑–∞–º–µ—Ä–∞ (–Ω–∞ –≤–µ—Ä—à–∏–Ω–µ –∏ –≤–æ –≤–ø–∞–¥–∏–Ω–µ), –∏–Ω–∞—á–µ –º–∞—Ç–µ–º–∞—Ç–∏–∫–∞ –Ω–µ —Å–º–æ–∂–µ—Ç –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–∏—Ç—å –µ—ë.

–ï—Å–ª–∏ –Ω–∞—Ä—É—à–∏—Ç—å —ç—Ç–æ –ø—Ä–∞–≤–∏–ª–æ, –≤–æ–∑–Ω–∏–∫–Ω–µ—Ç —Ñ–∏–∑–∏—á–µ—Å–∫–∏–π –∫–æ–ª–ª–∞–ø—Å ‚Äî **–ê–ª–∏–∞—Å–∏–Ω–≥ (Aliasing)**. –ß–∞—Å—Ç–æ—Ç–∞, –ø—Ä–µ–≤—ã—à–∞—é—â–∞—è –ü—Ä–µ–¥–µ–ª –ù–∞–π–∫–≤–∏—Å—Ç–∞ (`F_s / 2`), –ø—Ä–æ—Å–∫–æ–ª—å–∑–Ω–µ—Ç —Å–∫–≤–æ–∑—å –∑–∞–º–µ—Ä—ã –ê–¶–ü –∏ –ø—Ä–∏—Ç–≤–æ—Ä–∏—Ç—Å—è –Ω–∏–∑–∫–æ—á–∞—Å—Ç–æ—Ç–Ω—ã–º —Å–∏–≥–Ω–∞–ª–æ–º-–ø—Ä–∏–∑—Ä–∞–∫–æ–º. –û–Ω–∞ "–æ—Ç—Ä–∞–∑–∏—Ç—Å—è" –æ—Ç –∫—Ä–∞—è —Å–ø–µ–∫—Ç—Ä–∞ –∏ –Ω–∞–ª–æ–∂–∏—Ç—Å—è –ø–æ–≤–µ—Ä—Ö –ø–æ–ª–µ–∑–Ω–æ–≥–æ –∑–≤—É–∫–∞ –∫–∞–∫ –∂–µ—Å—Ç–∫–∏–π –∏—Å–∫–∞–∂–µ–Ω–Ω—ã–π —à—É–º.

## 6. –ö–≤–∞–¥—Ä–∞—Ç—É—Ä–Ω–∞—è –¥–∏—Å–∫—Ä–µ—Ç–∏–∑–∞—Ü–∏—è (–ó–∞–≥–∞–¥–∫–∞ I/Q –¥–∞–Ω–Ω—ã—Ö)

–°–≤–∏—Å—Ç–æ–∫ RTL-SDR –≤—ã–¥–∞–µ—Ç –ø–æ–ª–æ—Å—É —à–∏—Ä–∏–Ω–æ–π 2 –ú–ì—Ü, –ø–µ—Ä–µ–¥–∞–≤–∞—è –≤ –∫–æ–º–ø—å—é—Ç–µ—Ä –≤—Å–µ–≥–æ 2 –ú–µ–≥–∞-—Å—ç–º–ø–ª–∞ –≤ —Å–µ–∫—É–Ω–¥—É. –ù–æ —Ç–µ–æ—Ä–µ–º–∞ –ö–æ—Ç–µ–ª—å–Ω–∏–∫–æ–≤–∞ —Ç—Ä–µ–±—É–µ—Ç –º–∏–Ω–∏–º—É–º 4 MSPS! –ö–∞–∫ –æ–Ω–∏ –æ–±–æ—à–ª–∏ –∑–∞–∫–æ–Ω—ã —Ñ–∏–∑–∏–∫–∏?
–û—Ç–≤–µ—Ç –∫—Ä–æ–µ—Ç—Å—è –≤ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ **–ö–æ–º–ø–ª–µ–∫—Å–Ω—ã—Ö —á–∏—Å–µ–ª (I/Q –¥–∞–Ω–Ω—ã—Ö)**.

### –ü—Ä–æ–±–ª–µ–º–∞ –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ –Ω–∞–ø—Ä—è–∂–µ–Ω–∏—è (1D)
–ï—Å–ª–∏ –ê–¶–ü –∏–∑–º–µ—Ä—è–µ—Ç —Ç–æ–ª—å–∫–æ –æ–¥–Ω–æ –Ω–∞–ø—Ä—è–∂–µ–Ω–∏–µ: `U(t) = cos(2*œÄ * f * t)`. 
–ï—Å–ª–∏ –º—ã –ø–æ—Å—Ç–∞–≤–∏–º –≤—Ä–µ–º—è –Ω–∞ –ø–∞—É–∑—É (—Å–Ω–∏–º–æ–∫ –ê–¶–ü), –º—ã –ø–æ–ª—É—á–∏–º —á–∏—Å–ª–æ "0.7 –í–æ–ª—å—Ç". –ì–ª—è–¥—è –Ω–∞ —ç—Ç—É —Ç–æ—á–∫—É, –≤—ã –Ω–µ –∑–Ω–∞–µ—Ç–µ ‚Äî –≤–æ–ª–Ω–∞ —Å–µ–π—á–∞—Å —Ä–∞—Å—Ç–µ—Ç –∫ 1.0 –∏–ª–∏ –ø–∞–¥–∞–µ—Ç –∫ 0.0? –ú—ã –ø–æ—Ç–µ—Ä—è–ª–∏ **–≤–µ–∫—Ç–æ—Ä –≤—Ä–∞—â–µ–Ω–∏—è —Ñ–∞–∑—ã**.

–ë–æ–ª–µ–µ —Ç–æ–≥–æ, –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–π –∫–æ—Å–∏–Ω—É—Å —è–≤–ª—è–µ—Ç—Å—è —Å—É–º–º–æ–π –≤—Ä–∞—â–µ–Ω–∏–π –≤ *–¥–≤–µ –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω—ã–µ —Å—Ç–æ—Ä–æ–Ω—ã*:
`cos(x) = 0.5 * (e^(j*x) + e^(-j*x))`
–°—Ç–∞–Ω—Ü–∏—è, —Å–¥–≤–∏–Ω—É—Ç–∞—è –Ω–∞ `+100 –∫–ì—Ü` –≤—ã—à–µ –Ω—É–ª—è, –∏ —Å—Ç–∞–Ω—Ü–∏—è –Ω–∞ `-100 –∫–ì—Ü` –Ω–∏–∂–µ –Ω—É–ª—è —Å–æ–∑–¥–∞–¥—É—Ç –∞–±—Å–æ–ª—é—Ç–Ω–æ –∏–¥–µ–Ω—Ç–∏—á–Ω—ã–µ 1D –º–∞—Å—Å–∏–≤—ã —á–∏—Å–µ–ª. –û–Ω–∏ –Ω–∞–ª–æ–∂–∞—Ç—Å—è –¥—Ä—É–≥ –Ω–∞ –¥—Ä—É–≥–∞ (–∑–µ—Ä–∫–∞–ª—å–Ω—ã–π –∫–∞–Ω–∞–ª).

### –ö–≤–∞–¥—Ä–∞—Ç—É—Ä–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ (2D)
–ú—ã –±–µ—Ä–µ–º —Å–∏–≥–Ω–∞–ª —Å —Ç—é–Ω–µ—Ä–∞, –∫–æ–ø–∏—Ä—É–µ–º –µ–≥–æ –Ω–∞–¥–≤–æ–µ –∏ –ø—Ä–æ–ø—É—Å–∫–∞–µ–º —á–µ—Ä–µ–∑ **–î–í–ê** –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω—ã—Ö —Å–º–µ—Å–∏—Ç–µ–ª—è –∏ –ê–¶–ü. 
–ù–æ –≥–µ—Ç–µ—Ä–æ–¥–∏–Ω –Ω–∞ –≤—Ç–æ—Ä–æ–º —Å–º–µ—Å–∏—Ç–µ–ª–µ —Ñ–∏–∑–∏—á–µ—Å–∫–∏ —Å–¥–≤–∏–Ω—É—Ç —Ä–æ–≤–Ω–æ –Ω–∞ 90 –≥—Ä–∞–¥—É—Å–æ–≤ (—Å–∏–Ω—É—Å –≤–º–µ—Å—Ç–æ –∫–æ—Å–∏–Ω—É—Å–∞):
* **–ö–∞–Ω–∞–ª I (In-phase, –°–∏–Ω—Ñ–∞–∑–Ω—ã–π):** –£–º–Ω–æ–∂–∞–µ–º –Ω–∞ `cos(œâ*t)`.
* **–ö–∞–Ω–∞–ª Q (Quadrature, –ö–≤–∞–¥—Ä–∞—Ç—É—Ä–Ω—ã–π):** –£–º–Ω–æ–∂–∞–µ–º –Ω–∞ `-sin(œâ*t)`.

–ê–¶–ü –≤—ã–¥–∞–µ—Ç –ø–∞—Ä—ã —á–∏—Å–µ–ª: `[I, Q]`. –ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ —ç—Ç–æ –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç—ã —Ç–æ—á–∫–∏ –Ω–∞ –¥–µ–∫–∞—Ä—Ç–æ–≤–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏!
–ú—ã —Å–æ–±–∏—Ä–∞–µ–º –∏—Ö –≤ –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–µ —á–∏—Å–ª–æ: `Z = I + j*Q`.

**–§–æ—Ä–º—É–ª–∞ –≠–π–ª–µ—Ä–∞ –≥–ª–∞—Å–∏—Ç:** `e^(j*x) = cos(x) + j*sin(x)`.
–ù–∞—à 2D —Å–∏–≥–Ω–∞–ª (–ê–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–∏–π —Å–∏–≥–Ω–∞–ª) –æ–ø–∏—Å—ã–≤–∞–µ—Ç –∫–æ–Ω–µ—Ü –≤–µ–∫—Ç–æ—Ä–∞, –≤—Ä–∞—â–∞—é—â–µ–≥–æ—Å—è –ø–æ –∫—Ä—É–≥—É –Ω–∞ –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏!
* –í–µ–∫—Ç–æ—Ä –∫—Ä—É—Ç–∏—Ç—Å—è –ø—Ä–æ—Ç–∏–≤ —á–∞—Å–æ–≤–æ–π ‚Üí —á–∞—Å—Ç–æ—Ç–∞ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–∞—è.
* –í–µ–∫—Ç–æ—Ä –∫—Ä—É—Ç–∏—Ç—Å—è –ø–æ —á–∞—Å–æ–≤–æ–π ‚Üí —á–∞—Å—Ç–æ—Ç–∞ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–∞—è.

–ú—ã –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–∏–ª–∏ –≤–µ–∫—Ç–æ—Ä –≤—Ä–∞—â–µ–Ω–∏—è! –¢–µ–ø–µ—Ä—å –º—ã –∏–¥–µ–∞–ª—å–Ω–æ —Ä–∞–∑–ª–∏—á–∞–µ–º –ª–µ–≤—É—é –∏ –ø—Ä–∞–≤—É—é —Å—Ç–æ—Ä–æ–Ω—ã —Å–ø–µ–∫—Ç—Ä–∞ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –Ω—É–ª—è, —É–¥–≤–∞–∏–≤–∞—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—É—é –ø–æ–ª–æ—Å—É –ø—Ä–æ–ø—É—Å–∫–∞–Ω–∏—è –ê–¶–ü –±–µ–∑ –ø–æ–≤—ã—à–µ–Ω–∏—è –µ–≥–æ —á–∞—Å—Ç–æ—Ç—ã.

```python
# –ß—Ç–µ–Ω–∏–µ –±–∞–π—Ç–æ–≤ RTL_TCP ‚Äî –±–∞–π—Ç—ã –≤—Å–µ–≥–¥–∞ –∏–¥—É—Ç –ø–∞—Ä–∞–º–∏ [I, Q]
# 8-–±–∏—Ç–Ω—ã–π –±–µ–∑–∑–Ω–∞–∫–æ–≤—ã–π —Ñ–æ—Ä–º–∞—Ç: 0..255 (127.5 ‚Äî —ç—Ç–æ 0 –í–æ–ª—å—Ç)

complex_baseband = []
n_pairs = len(raw_bytes) // 2

for k in range(n_pairs):
    # –ü—Ä–∏–≤–æ–¥–∏–º –¥–∏–∞–ø–∞–∑–æ–Ω –∫ [-127.5, +127.5]
    i_float = raw_bytes[k*2]     - 127.5  
    q_float = raw_bytes[k*2 + 1] - 127.5
    
    # –¢–µ–ø–µ—Ä—å —ç—Ç–æ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∞—è –≤—Ä–∞—â–∞—é—â–∞—è—Å—è —Ç–æ—á–∫–∞ Z
    vector = complex(i_float, q_float)
    complex_baseband.append(vector)
```

## 7. –¶–∏—Ñ—Ä–æ–≤–∞—è —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏—è –∏ –¥–µ—Ü–∏–º–∞—Ü–∏—è

–ó–≤—É–∫–æ–≤–æ–π –∫–∞—Ä—Ç–µ —Ç—Ä–µ–±—É–µ—Ç—Å—è –∞—É–¥–∏–æ–ø–æ—Ç–æ–∫ `48 000 SPS`. RTL-SDR —à–ª–µ—Ç `2 400 000 SPS`. –ù–∞–º –Ω—É–∂–Ω–æ –ø—Ä–æ—Ä–µ–¥–∏—Ç—å –º–∞—Å—Å–∏–≤ –≤ 50 —Ä–∞–∑ (**–î–µ—Ü–∏–º–∞—Ü–∏—è**).

–ï—Å–ª–∏ –ø—Ä–æ—Å—Ç–æ —Å–¥–µ–ª–∞—Ç—å —Å—Ä–µ–∑ `array[::50]`, –º—ã –º–≥–Ω–æ–≤–µ–Ω–Ω–æ –Ω–∞—Ä—É—à–∏–º —Ç–µ–æ—Ä–µ–º—É –ö–æ—Ç–µ–ª—å–Ω–∏–∫–æ–≤–∞ (–Ω–æ–≤—ã–º –ø—Ä–µ–¥–µ–ª–æ–º –ù–∞–π–∫–≤–∏—Å—Ç–∞ —Å—Ç–∞–Ω–µ—Ç 24 –∫–ì—Ü). –í–µ—Å—å –≤—ã—Å–æ–∫–æ—á–∞—Å—Ç–æ—Ç–Ω—ã–π —Ä–∞–¥–∏–æ—ç—Ñ–∏—Ä —Ä—É—Ö–Ω–µ—Ç –≤–Ω–∏–∑ –≤ –≤–∏–¥–µ –∞–ª–∏–∞—Å–∏–Ω–≥–æ–≤–æ–≥–æ —à—É–º–∞.
–ó–Ω–∞—á–∏—Ç, —Å–Ω–∞—á–∞–ª–∞ –Ω—É–∂–µ–Ω **–¶–∏—Ñ—Ä–æ–≤–æ–π –§–∏–ª—å—Ç—Ä –ù–∏–∂–Ω–∏—Ö –ß–∞—Å—Ç–æ—Ç (LPF)**, —É–±–∏–≤–∞—é—â–∏–π –≤—Å—ë –≤—ã—à–µ 24 –∫–ì—Ü –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏.

### FIR –ø—Ä–æ—Ç–∏–≤ IIR —Ñ–∏–ª—å—Ç—Ä–æ–≤
* **–ö–ò–• / FIR (–° –∫–æ–Ω–µ—á–Ω–æ–π –∏–º–ø—É–ª—å—Å–Ω–æ–π —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–æ–π):** –≠—Ç–æ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∞—è —Å–≤–µ—Ä—Ç–∫–∞ (—Å–∫–æ–ª—å–∑—è—â–µ–µ –Ω–∞–ª–æ–∂–µ–Ω–∏–µ —Ñ–æ—Ä–º—ã —Ñ–∏–ª—å—Ç—Ä–∞ –Ω–∞ —Å–∏–≥–Ω–∞–ª). –ò–¥–µ–∞–ª—å–Ω—ã, —Ç–∞–∫ –∫–∞–∫ –æ–±–ª–∞–¥–∞—é—Ç –ª–∏–Ω–µ–π–Ω–æ–π —Ñ–∞–∑–æ–π (–Ω–µ –∏—Å–∫–∞–∂–∞—é—Ç —Ñ–æ—Ä–º—É –≤–æ–ª–Ω—ã). –ù–æ —Ç—Ä–µ–±—É—é—Ç –æ–≥—Ä–æ–º–Ω–æ–≥–æ —á–∏—Å–ª–∞ —É–º–Ω–æ–∂–µ–Ω–∏–π –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä–∞.
* **–ë–ò–• / IIR (–° –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ–π –∏–º–ø—É–ª—å—Å–Ω–æ–π —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–æ–π):** –ê–Ω–∞–ª–æ–≥ —Ä–µ–∞–ª—å–Ω–æ–≥–æ RC-–∫–æ–Ω—Ç—É—Ä–∞. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç –ø—Ä–æ—à–ª—ã–µ –≤—ã—Ö–æ–¥—ã (—Ä–µ–∫—É—Ä—Å–∏—è, –æ–±—Ä–∞—Ç–Ω–∞—è —Å–≤—è–∑—å): `y[n] = x[n] + a*y[n-1]`. –°—á–∏—Ç–∞–µ—Ç—Å—è –∑–∞ 1 —Ç–∞–∫—Ç, –Ω–æ –Ω–µ–ª–∏–Ω–µ–π–Ω–æ "—Ä–∞–∑–º–∞–∑—ã–≤–∞–µ—Ç" –∑–≤—É–∫.

### –ò–Ω—Ç–µ–≥—Ä–∞—Ç–æ—Ä —Å–∫–æ–ª—å–∑—è—â–µ–≥–æ —Å—Ä–µ–¥–Ω–µ–≥–æ (Box-car)
–î–ª—è –∏–¥–µ–∞–ª—å–Ω–æ–≥–æ –±–∞–ª–∞–Ω—Å–∞ —Å–∫–æ—Ä–æ—Å—Ç–∏ –≤ Python –º—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º –ö–ò–•-—Ñ–∏–ª—å—Ç—Ä —Ç–∏–ø–∞ `Box-car` ‚Äî –ø—Ä–æ—Å—Ç–æ–µ —Å—É–º–º–∏—Ä–æ–≤–∞–Ω–∏–µ `N` —Å–æ—Å–µ–¥–Ω–∏—Ö —Å—ç–º–ø–ª–æ–≤ –±–µ–∑ –≤–µ—Å–æ–≤.
–ú–∞—Ç–µ–º–∞—Ç–∏–∫–∞ –§—É—Ä—å–µ –¥–æ–∫–∞–∑—ã–≤–∞–µ—Ç, —á—Ç–æ —Å–≤–µ—Ä—Ç–∫–∞ —Å –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω—ã–º –æ–∫–Ω–æ–º –≤–æ –≤—Ä–µ–º–µ–Ω–∏ –∏–¥–µ–Ω—Ç–∏—á–Ω–∞ —É–º–Ω–æ–∂–µ–Ω–∏—é —Å–ø–µ–∫—Ç—Ä–∞ –Ω–∞ —Ñ—É–Ω–∫—Ü–∏—é `sinc(x) = sin(x)/x`. –§—É–Ω–∫—Ü–∏—è `sinc` –ø–∞–¥–∞–µ—Ç –¥–æ —á–∏—Å—Ç–æ–≥–æ –Ω—É–ª—è —Ä–æ–≤–Ω–æ –Ω–∞ —á–∞—Å—Ç–æ—Ç–∞—Ö, –∫—Ä–∞—Ç–Ω—ã—Ö —à–∞–≥—É —É—Å—Ä–µ–¥–Ω–µ–Ω–∏—è. –≠—Ç–æ –∏–¥–µ–∞–ª—å–Ω—ã–π –∞–Ω—Ç–∏–∞–ª–∏–∞—Å–∏–Ω–≥–æ–≤—ã–π —Ñ–∏–ª—å—Ç—Ä, –Ω–µ —Ç—Ä–µ–±—É—é—â–∏–π –≤—ã—á–∏—Å–ª–µ–Ω–∏—è —Å–∏–Ω—É—Å–æ–≤!

```python
# –û–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è FIR-—Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏—è (sinc) –∏ –¥–µ—Ü–∏–º–∞—Ü–∏—è –≤ 5 —Ä–∞–∑
n_decimated = len(i_channel) - (len(i_channel) % 5)
inv_factor = 1.0 / 5.0

audio_baseband = [
    # –£—Å—Ä–µ–¥–Ω—è–µ–º 5 —Ç–æ—á–µ–∫ (Low-Pass Filter, —É–±–∏–≤–∞–µ—Ç —à—É–º)
    (i_channel[j] + i_channel[j+1] + i_channel[j+2] + i_channel[j+3] + i_channel[j+4]) * inv_factor
    for j in range(0, n_decimated, 5)  # –®–∞–≥–∞–µ–º –ø–æ 5 (–î–µ—Ü–∏–º–∞—Ü–∏—è)
]
```

## 8. –ê–º–ø–ª–∏—Ç—É–¥–Ω–∞—è –º–æ–¥—É–ª—è—Ü–∏—è (AM)

**–§–∏–∑–∏–∫–∞:** 
–ü–µ—Ä–µ–¥–∞—Ç—á–∏–∫ –æ—Å—Ç–∞–≤–ª—è–µ—Ç —á–∞—Å—Ç–æ—Ç—É –ø–æ—Å—Ç–æ—è–Ω–Ω–æ–π, –Ω–æ –º–µ–Ω—è–µ—Ç –Ω–∞–ø—Ä—è–∂–µ–Ω–∏–µ –∏–∑–ª—É—á–∞—Ç–µ–ª—è (–ê–º–ø–ª–∏—Ç—É–¥—É) –≤ —Ç–∞–∫—Ç —Å –≥–æ–ª–æ—Å–æ–º –¥–∏–∫—Ç–æ—Ä–∞ `m(t)`.
`U_AM(t) = [A_0 + m(t)] * cos(2*œÄ * f_c * t)`

–ü—Ä–∏ —Ä–∞–∑–ª–æ–∂–µ–Ω–∏–∏ –§—É—Ä—å–µ —ç—Ç–æ —É–º–Ω–æ–∂–µ–Ω–∏–µ –ø–æ—Ä–æ–∂–¥–∞–µ—Ç —Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω—ã–π –ø–∏–∫ –Ω–µ—Å—É—â–µ–π `A_0` –∏ –¥–≤–µ —Å–∏–º–º–µ—Ç—Ä–∏—á–Ω—ã–µ –∫–æ–ø–∏–∏ –∞—É–¥–∏–æ—Å–ø–µ–∫—Ç—Ä–∞: USB (—Å–≤–µ—Ä—Ö—É) –∏ LSB (—Å–Ω–∏–∑—É). –≠—Ç–æ –±–µ–∑—É–º–Ω–æ –Ω–µ—ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ –ø–æ —ç–Ω–µ—Ä–≥–∏–∏.

**–ú–∞—Ç–µ–º–∞—Ç–∏–∫–∞ –ê–ú –î–µ—Ç–µ–∫—Ç–æ—Ä–∞:**
–ü–æ—Å–ª–µ —Ç–æ–≥–æ –∫–∞–∫ —Ç—é–Ω–µ—Ä –ø–µ—Ä–µ–Ω–µ—Å —ç—Ç–æ—Ç —Å–∏–≥–Ω–∞–ª –≤ `Baseband` (0 –ì—Ü), –Ω–µ—Å—É—â–∞—è —Å—Ç–∞–ª–∞ –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–º —Ç–æ–∫–æ–º (DC).
–ü–æ—Å–∫–æ–ª—å–∫—É –∑–≤—É–∫ –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω –∏—Å–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ –≤ —Å–∏–ª–µ —Å–∏–≥–Ω–∞–ª–∞, –Ω–∞—Å –Ω–µ –∏–Ω—Ç–µ—Ä–µ—Å—É–µ—Ç —É–≥–æ–ª –≤—Ä–∞—â–µ–Ω–∏—è I/Q –≤–µ–∫—Ç–æ—Ä–∞. –ù–∞–º –Ω—É–∂–Ω–∞ —Ç–æ–ª—å–∫–æ –µ–≥–æ **–≥–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–∞—è –¥–ª–∏–Ω–∞** –∏–∑ —Ü–µ–Ω—Ç—Ä–∞ –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç.

–ê–ª–≥–æ—Ä–∏—Ç–º (–î–µ—Ç–µ–∫—Ç–æ—Ä –û–≥–∏–±–∞—é—â–µ–π): –¢–µ–æ—Ä–µ–º–∞ –ü–∏—Ñ–∞–≥–æ—Ä–∞ `sqrt(I^2 + Q^2)`.

```python
audio_env = []
for k in range(n_pairs):
    i = raw[k*2] - 127.5
    q = raw[k*2+1] - 127.5
    
    # –ì–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–∞—è –¥–ª–∏–Ω–∞ –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–≥–æ —á–∏—Å–ª–∞ (–ê–º–ø–ª–∏—Ç—É–¥–∞ –≤–æ–ª–Ω—ã)
    magnitude = math.sqrt(i**2 + q**2)
    audio_env.append(magnitude)

# –ü–æ—Å–∫–æ–ª—å–∫—É A_0 –≤—Å–µ–≥–¥–∞ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ (–≤–æ–ª–Ω–∞ –Ω–µ –¥–æ–ª–∂–Ω–∞ –ø–µ—Ä–µ–≤–æ—Ä–∞—á–∏–≤–∞—Ç—å—Å—è),
# –†–µ–∑—É–ª—å—Ç–∞—Ç —Å–æ–¥–µ—Ä–∂–∏—Ç –≥–∏–≥–∞–Ω—Ç—Å–∫–æ–µ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ DC-—Å–º–µ—â–µ–Ω–∏–µ (–ü–æ—Å—Ç–æ—è–Ω–Ω—ã–π —Ç–æ–∫).
```

## 9. –û–¥–Ω–æ–ø–æ–ª–æ—Å–Ω–∞—è –º–æ–¥—É–ª—è—Ü–∏—è (SSB)

**–§–∏–∑–∏–∫–∞:**
SSB - —ç—Ç–æ –≤–µ–Ω–µ—Ü —ç—Ñ–∏—Ä–Ω–æ–π —Ä–∞–¥–∏–æ—Ç–µ—Ö–Ω–∏–∫–∏. –í AM –Ω–µ—Å—É—â–∞—è —Å–æ–¥–µ—Ä–∂–∏—Ç 0 –±–∏—Ç –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏, –Ω–æ —Å–∂–∏—Ä–∞–µ—Ç 66% –º–æ—â–Ω–æ—Å—Ç–∏ —Ä–∞–¥–∏–æ—Å—Ç–∞–Ω—Ü–∏–∏. –î–≤–µ –±–æ–∫–æ–≤—ã–µ –ø–æ–ª–æ—Å—ã –∞–±—Å–æ–ª—é—Ç–Ω–æ –∑–µ—Ä–∫–∞–ª—å–Ω—ã ‚Äî –∑–∞—á–µ–º –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å –æ–±–µ –∏ –∑–∞–Ω–∏–º–∞—Ç—å 10 –∫–ì—Ü —ç—Ñ–∏—Ä–∞?
–ü–µ—Ä–µ–¥–∞—Ç—á–∏–∫ SSB –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ —É–º–Ω–æ–∂–∞–µ—Ç –∑–≤—É–∫ –Ω–∞ –∞–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–∏–µ —Ñ–∏–ª—å—Ç—Ä—ã, –ø–æ–ª–Ω–æ—Å—Ç—å—é —É–Ω–∏—á—Ç–æ–∂–∞—è –Ω–µ—Å—É—â—É—é (Suppressed Carrier) –∏ –ø–æ–¥–∞–≤–ª—è—è –æ–¥–Ω—É –∏–∑ –ø–æ–ª–æ—Å. –í —ç—Ñ–∏—Ä —É—Ö–æ–¥–∏—Ç —É–∑–∫–∏–π —à–ª–µ–π—Ñ –≤ 3 –∫–ì—Ü. –í—Å—è —ç–Ω–µ—Ä–≥–∏—è –∏–¥–µ—Ç –≤ –≥–æ–ª–æ—Å.
* **USB:** $1$ –∫–ì—Ü –∑–≤—É–∫ —É–ª–µ—Ç–∞–µ—Ç –Ω–∞ $(F_{—Å–µ—Ä–µ–¥–∏–Ω–∞} + 1\text{ –∫–ì—Ü})$.

**–ú–∞—Ç–µ–º–∞—Ç–∏–∫–∞ SSB –î–µ–º–æ–¥—É–ª—è—Ç–æ—Ä–∞ (–°–ø–µ–∫—Ç—Ä–∞–ª—å–Ω–æ–µ –°–≤–æ—Ä–∞—á–∏–≤–∞–Ω–∏–µ):**
–ö–∞–∫ –ø—Ä–∏–Ω—è—Ç—å —ç—Ç–æ—Ç –∞—Å–∫–µ—Ç–∏—á–Ω—ã–π –∫—É—Å–æ–∫ —Å–ø–µ–∫—Ç—Ä–∞ –±–µ–∑ –Ω–µ—Å—É—â–µ–π?

–ï—Å–ª–∏ SDR –Ω–∞—Å—Ç—Ä–æ–µ–Ω —Ç–∞–∫, —á—Ç–æ —Ü–µ–Ω—Ç—Ä —Å–º–µ—â–µ–Ω —Ç–æ—á–Ω–æ –≤ —É–¥–∞–ª–µ–Ω–Ω—É—é –Ω–µ—Å—É—â—É—é, —Ç–æ SSB —Å–ø–µ–∫—Ç—Ä –∏–¥–µ–∞–ª—å–Ω–æ –ª–æ–∂–∏—Ç—Å—è –≤ –∫–æ–º–ø–ª–µ–∫—Å–Ω—ã–π `Baseband`. 
USB –∂–∏–≤–µ—Ç *—Ç–æ–ª—å–∫–æ* –Ω–∞ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö —á–∞—Å—Ç–æ—Ç–∞—Ö. LSB ‚Äî *—Ç–æ–ª—å–∫–æ* –Ω–∞ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã—Ö.

–ó–≤—É–∫ –Ω–∞ –∫–æ–ª–æ–Ω–∫–∞—Ö ‚Äî —ç—Ç–æ 1D –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–π –º–∞—Å—Å–∏–≤. –ü–æ –∑–∞–∫–æ–Ω–∞–º –º–∞—Ç–µ–º–∞—Ç–∏–∫–∏, –æ–¥–Ω–æ–º–µ—Ä–Ω—ã–π —Å–∏–≥–Ω–∞–ª –≤—Å–µ–≥–¥–∞ —Å–∏–º–º–µ—Ç—Ä–∏—á–µ–Ω (–ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–µ —á–∞—Å—Ç–æ—Ç—ã –æ—Ç–±—Ä–∞—Å—ã–≤–∞—é—Ç –∑–µ—Ä–∫–∞–ª—å–Ω—É—é —Ç–µ–Ω—å –Ω–∞ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ –ø–æ–ª—É–ø–ª–æ—Å–∫–æ—Å—Ç–∏).

**–§–æ–∫—É—Å:** –ß—Ç–æ–±—ã –ø—Ä–µ–≤—Ä–∞—Ç–∏—Ç—å –ª–µ–≤–æ—Å—Ç–æ—Ä–æ–Ω–Ω–∏–π LSB (–∏–ª–∏ –ø—Ä–∞–≤–æ—Å—Ç–æ—Ä–æ–Ω–Ω–∏–π USB) –≤–µ–∫—Ç–æ—Ä–Ω—ã–π —Å–ø–µ–∫—Ç—Ä –≤ —Å–∏–º–º–µ—Ç—Ä–∏—á–Ω—ã–π —Å–ª—ã—à–∏–º—ã–π –∑–≤—É–∫, –Ω—É–∂–Ω–æ –ø—Ä–æ—Å—Ç–æ **–≤—ã—Ä–µ–∑–∞—Ç—å –º–Ω–∏–º—É—é –æ—Å—å (Q)**.
–û–ø–µ—Ä–∞—Ü–∏—è `Real(Z)` –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ —É–Ω–∏—á—Ç–æ–∂–∞–µ—Ç —Å–ø–æ—Å–æ–±–Ω–æ—Å—Ç—å I/Q —Å–∏–≥–Ω–∞–ª–∞ —Ä–∞–∑–ª–∏—á–∞—Ç—å –ø–ª—é—Å –∏ –º–∏–Ω—É—Å. –õ–µ–≤–∞—è –ø–æ–ª—É–ø–ª–æ—Å–∫–æ—Å—Ç—å –º–≥–Ω–æ–≤–µ–Ω–Ω–æ –¥–µ–≥—Ä–∞–¥–∏—Ä—É–µ—Ç, –æ—Ç—Ä–∞–∂–∞–µ—Ç—Å—è –≤–¥–æ–ª—å Y-–æ—Å–∏ –∏ —Å—Ö–ª–æ–ø—ã–≤–∞–µ—Ç—Å—è –≤ –Ω—É–∂–Ω—ã–π –Ω–∞–º 1D –∑–≤—É–∫–æ–≤–æ–π –º–∞—Å—Å–∏–≤. –§–∞–∑—ã –≤—ã—Å—Ç—Ä–∞–∏–≤–∞—é—Ç—Å—è –≤ —Ä—è–¥!

```python
# –ò—Å—Ç–∏–Ω–Ω—ã–π –ê–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–∏–π SSB –î–µ–º–æ–¥—É–ª—è—Ç–æ—Ä
# –í –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–º baseband –∫–∞–Ω–∞–ª I ‚Äî —ç—Ç–æ –£–ñ–ï –≥–æ—Ç–æ–≤—ã–π 1D –∑–≤—É–∫!
# –ù–∞–º –Ω–µ –Ω—É–∂–Ω—ã –Ω–∏ –¥–µ—Ç–µ–∫—Ç–æ—Ä—ã –æ–≥–∏–±–∞—é—â–µ–π, –Ω–∏ BFO (–ì–µ—Ç–µ—Ä–æ–¥–∏–Ω—ã –±–∏–µ–Ω–∏–π).

for k in range(n_pairs):
    audio[k] = raw[k*2] - 127.5 # –ü—Ä–æ—Å—Ç–æ –∏–≥–Ω–æ—Ä–∏—Ä—É–µ–º —Å—ã—Ä–æ–π –º–∞—Å—Å–∏–≤ Q.
```

## 10. –ß–∞—Å—Ç–æ—Ç–Ω–∞—è –º–æ–¥—É–ª—è—Ü–∏—è (FM)

**–§–∏–∑–∏–∫–∞:**
–í AM –º–æ–ª–Ω–∏—è ‚Äî —ç—Ç–æ –ø—Ä—è–º–æ–π —Ñ–∏–∑–∏—á–µ—Å–∫–∏–π —Å–∫–∞—á–æ–∫ –Ω–∞–ø—Ä—è–∂–µ–Ω–∏—è (–º–æ—â–Ω—ã–π —â–µ–ª—á–æ–∫ –≤ –¥–∏–Ω–∞–º–∏–∫–µ). –í FM –∞–º–ø–ª–∏—Ç—É–¥–∞ –ø–µ—Ä–µ–¥–∞—Ç—á–∏–∫–∞ –∑–∞—Ü–µ–º–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–∞ (–æ–Ω–∞ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞). –ó–≤—É–∫ `m(t)` –º–µ–Ω—è–µ—Ç **—á–∞—Å—Ç–æ—Ç—É –∫–æ–ª–µ–±–∞–Ω–∏–π** –Ω–µ—Å—É—â–µ–π.
–ú–≥–Ω–æ–≤–µ–Ω–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞ ‚Äî —ç—Ç–æ —Å–∫–æ—Ä–æ—Å—Ç—å –Ω–∞–∫—Ä—É—á–∏–≤–∞–Ω–∏—è —Ñ–∞–∑—ã:
`U_FM(t) = A_0 * cos(2*œÄ * f_c * t + 2*œÄ * k_f * ‚à´m(œÑ)dœÑ)`
–ò–Ω—Ç–µ–≥—Ä–∞–ª –∑–≤—É–∫–∞ —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è —Ñ–∞–∑–æ–π! –ü–æ–º–µ—Ö–∏ –Ω–µ –º–æ–≥—É—Ç –±—ã—Å—Ç—Ä–æ –º–µ–Ω—è—Ç—å —á–∞—Å—Ç–æ—Ç—É —Ä–∞–¥–∏–æ–≤–æ–ª–Ω—ã, –ø–æ—ç—Ç–æ–º—É FM –∑–≤—É—á–∏—Ç –∫—Ä–∏—Å—Ç–∞–ª—å–Ω–æ —á–∏—Å—Ç–æ.

**–ú–∞—Ç–µ–º–∞—Ç–∏–∫–∞ FM –î–∏—Å–∫—Ä–∏–º–∏–Ω–∞—Ç–æ—Ä–∞ (–ü–æ–ª—è—Ä–Ω—ã–π –¥–µ—Ç–µ–∫—Ç–æ—Ä):**
–ß–∞—Å—Ç–æ—Ç–∞ ‚Äî —ç—Ç–æ **–ø–µ—Ä–≤–∞—è –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è —Ñ–∞–∑—ã –ø–æ –≤—Ä–µ–º–µ–Ω–∏**: `f(t) = (1 / 2*œÄ) * dœÜ/dt`.

–ó–Ω–∞—á–∏—Ç, –Ω–∞–º –Ω—É–∂–Ω–æ:
1. –í—ã—á–∏—Å–ª–∏—Ç—å —Ñ–∞–∑—É –∫–∞–∂–¥–æ–≥–æ 2D –≤–µ–∫—Ç–æ—Ä–∞: `œÜ = atan2(Q, I)`
2. –ù–∞–π—Ç–∏ —Å–∫–æ—Ä–æ—Å—Ç—å –µ—ë –∏–∑–º–µ–Ω–µ–Ω–∏—è: `ŒîœÜ = œÜ[n] - œÜ[n-1]`

–ó–∞–≥–≤–æ–∑–¥–∫–∞ –≤ —Ç–æ–º, —á—Ç–æ `atan2` –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —É–≥–ª—ã –æ—Ç `-œÄ` –¥–æ `+œÄ`. –í—ã—á–∏—Ç–∞–Ω–∏–µ —É–≥–ª–∞ `-179¬∞` –∏–∑ `+179¬∞` –¥–∞—Å—Ç –ø–µ—Ä–µ—Å–∫–æ–∫ –≤ `+358¬∞`, —Ö–æ—Ç—è —Ä–µ–∞–ª—å–Ω—ã–π –ø—É—Ç—å –≤–µ–∫—Ç–æ—Ä–∞ —Å–æ—Å—Ç–∞–≤–∏–ª –≤—Å–µ–≥–æ `-2¬∞`. –≠—Ç–æ—Ç –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–π —Ä–∞–∑—Ä—ã–≤ –Ω–∞ –≥—Ä–∞–Ω–∏—Ü–µ –∫—Ä—É–≥–∞ –≤–∑–æ—Ä–≤–µ—Ç –∫–æ–ª–æ–Ω–∫–∏. –ù–∞–º –Ω—É–∂–Ω–∞ –∞–ª–≥–µ–±—Ä–∞–∏—á–µ—Å–∫–∞—è —Ä–∞–∑–Ω–æ—Å—Ç—å —É–≥–ª–æ–≤ –±–µ–∑ —Ç—Ä–∏–≥–æ–Ω–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–∏—Ö –ø–µ—Ä–µ—Å–∫–æ–∫–æ–≤.

**–¢—Ä—é–∫ –ö–æ–º–ø–ª–µ–∫—Å–Ω–æ–≥–æ –°–æ–ø—Ä—è–∂–µ–Ω–∏—è:**
–í –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–π –∞—Ä–∏—Ñ–º–µ—Ç–∏–∫–µ, –µ—Å–ª–∏ —É–º–Ω–æ–∂–∏—Ç—å –æ–¥–Ω–æ —á–∏—Å–ª–æ –Ω–∞ *–∫–æ–º–ø–ª–µ–∫—Å–Ω–æ-—Å–æ–ø—Ä—è–∂–µ–Ω–Ω–æ–µ* –æ—Ç –¥—Ä—É–≥–æ–≥–æ `Z1 * conj(Z2)`, –∏—Ö –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–µ —É–≥–ª—ã **–∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –±–µ–∑–æ–ø–∞—Å–Ω–æ –≤—ã—á—Ç—É—Ç—Å—è**, –∞ –¥–ª–∏–Ω—ã —É–º–Ω–æ–∂–∞—Ç—Å—è.
`angle(Z1 * conj(Z2)) = œÜ1 - œÜ2`. –ò –º—ã –≤—ã–∑–æ–≤–µ–º —Ç—è–∂–µ–ª—ã–π `atan2` –≤—Å–µ–≥–æ –æ–¥–∏–Ω —Ä–∞–∑ –¥–ª—è —Ñ–∏–Ω–∞–ª—å–Ω–æ–≥–æ –≤–µ–∫—Ç–æ—Ä–∞ —Ä–∞–∑–Ω–æ—Å—Ç–∏!

```python
# –®–∏—Ä–æ–∫–æ–ø–æ–ª–æ—Å–Ω—ã–π FM –¥–µ–º–æ–¥—É–ª—è—Ç–æ—Ä
prev_i, prev_q = 0.0, 0.0

for k in range(n_pairs):
    i_val = raw[k*2] - 127.5
    q_val = raw[k*2+1] - 127.5

    # –£–º–Ω–æ–∂–∞–µ–º —Ç–µ–∫—É—â–∏–π —Å—ç–º–ø–ª –Ω–∞ –°–û–ü–†–Ø–ñ–ï–ù–ù–´–ô –ø—Ä–µ–¥—ã–¥—É—â–∏–π —Å—ç–º–ø–ª
    # –ï—Å–ª–∏ Z2 = prev_i + j*prev_q, —Ç–æ conj(Z2) = prev_i - j*prev_q
    
    # –ê–ª–≥–µ–±—Ä–∞–∏—á–µ—Å–∫–æ–µ —Ä–∞—Å–∫—Ä—ã—Ç–∏–µ —Å–∫–∞–ª—è—Ä–Ω–æ–≥–æ –∏ –≤–µ–∫—Ç–æ—Ä–Ω–æ–≥–æ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏—è 2D —Ç–æ—á–µ–∫
    p_i = i_val * prev_i + q_val * prev_q  # –í—ã—á–∏—Å–ª—è–µ—Ç –∫–æ—Å–∏–Ω—É—Å —É–≥–ª–∞ (Dot product)
    p_q = q_val * prev_i - i_val * prev_q  # –í—ã—á–∏—Å–ª—è–µ—Ç —Å–∏–Ω—É—Å —É–≥–ª–∞ (Cross product)

    # –§—É–Ω–∫—Ü–∏—è math.atan2(sin, cos) –±–µ–∑–æ–ø–∞—Å–Ω–æ –≤–µ—Ä–Ω–µ—Ç –≥–æ—Ç–æ–≤—É—é –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—É—é [–æ—Ç -œÄ –¥–æ œÄ]
    # –≠—Ç–æ –∏ –µ—Å—Ç—å –æ—Ä–∏–≥–∏–Ω–∞–ª—å–Ω—ã–π –≥–æ–ª–æ—Å –¥–∏–∫—Ç–æ—Ä–∞!
    audio_fm[k] = math.atan2(p_q, p_i)
    
    # –ó–∞–ø–æ–º–∏–Ω–∞–µ–º —Ç–µ–∫—É—â–∏–π –≤–µ–∫—Ç–æ—Ä –¥–ª—è —Å–ª–µ–¥—É—é—â–µ–≥–æ —à–∞–≥–∞
    prev_i, prev_q = i_val, q_val
```

## –ò—Ç–æ–≥ –∫—É—Ä—Å–∞

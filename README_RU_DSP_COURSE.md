*   `œâ_c` ‚Äî –£–≥–ª–æ–≤–∞—è —á–∞—Å—Ç–æ—Ç–∞ `œâ_c = 2*œÄ * f_c` (–≤ —Ä–∞–¥–∏–∞–Ω–∞—Ö –≤ —Å–µ–∫—É–Ω–¥—É).
*   `f_c` ‚Äî –ù–µ—Å—É—â–∞—è —á–∞—Å—Ç–æ—Ç–∞ (–≤ –ì–µ—Ä—Ü–∞—Ö).
*   `œÜ` ‚Äî –ù–∞—á–∞–ª—å–Ω–∞—è —Ñ–∞–∑–∞ (–≤ —Ä–∞–¥–∏–∞–Ω–∞—Ö).

–ß—Ç–æ–±—ã –ø–µ—Ä–µ–¥–∞—Ç—å –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é (–∑–≤—É–∫ –∏–ª–∏ –¥–∞–Ω–Ω—ã–µ), –º—ã –¥–æ–ª–∂–Ω—ã –∏–∑–º–µ–Ω–∏—Ç—å (—Å–º–æ–¥—É–ª–∏—Ä–æ–≤–∞—Ç—å) –æ–¥–∏–Ω –∏–∑ —ç—Ç–∏—Ö —Ç—Ä–µ—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ (`A`, `œâ_c` –∏–ª–∏ `œÜ`) –ø–æ –∑–∞–∫–æ–Ω—É –Ω–∞—à–µ–≥–æ –ø–µ—Ä–µ–¥–∞–≤–∞–µ–º–æ–≥–æ —Å–∏–≥–Ω–∞–ª–∞ `m(t)`. 

–≠—Ñ–∏—Ä —Å–æ–¥–µ—Ä–∂–∏—Ç –º–∏–ª–ª–∏–æ–Ω—ã —Ç–∞–∫–∏—Ö –≤–æ–ª–Ω. –†–∞–∑–¥–µ–ª–∏—Ç—å –∏—Ö –≤–æ –≤—Ä–µ–º–µ–Ω–Ω–æ–π –æ–±–ª–∞—Å—Ç–∏ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ. –†–µ—à–µ–Ω–∏–µ –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ **–ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–∏ –§—É—Ä—å–µ**. –ú–∞—Ç–µ–º–∞—Ç–∏–∫–∞ –§—É—Ä—å–µ –¥–æ–∫–∞–∑—ã–≤–∞–µ—Ç, —á—Ç–æ –ª—é–±—É—é –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω—É—é –≤–æ –≤—Ä–µ–º–µ–Ω–∏ –∏ —ç–Ω–µ—Ä–≥–∏–∏ —Ñ—É–Ω–∫—Ü–∏—é –º–æ–∂–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç—å –∫–∞–∫ –∏–Ω—Ç–µ–≥—Ä–∞–ª –æ—Ç –∫–æ–º–ø–ª–µ–∫—Å–Ω—ã—Ö —ç–∫—Å–ø–æ–Ω–µ–Ω—Ç (–∏–ª–∏ –±–µ—Å–∫–æ–Ω–µ—á–Ω—É—é —Å—É–º–º—É —Å–∏–Ω—É—Å–æ–∏–¥). –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ $X(f) = \int_{-\infty}^{\infty} x(t) e^{-j2\pi ft} dt$ –ø–µ—Ä–µ–≤–æ–¥–∏—Ç –Ω–∞—à —Å–∏–≥–Ω–∞–ª –∏–∑ –≤—Ä–µ–º–µ–Ω–Ω–æ–π –≤ —á–∞—Å—Ç–æ—Ç–Ω—É—é –æ–±–ª–∞—Å—Ç—å, –≥–¥–µ –∫–∞–∂–¥–∞—è —Ä–∞–¥–∏–æ—Å—Ç–∞–Ω—Ü–∏—è –≤—ã–≥–ª—è–¥–∏—Ç –∫–∞–∫ –æ—Ç–¥–µ–ª—å–Ω—ã–π –∏–∑–æ–ª–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –ø–∏–∫ –≤ —Å–ø–µ–∫—Ç—Ä–µ.

## 3. –ê–Ω–∞–ª–æ–≥–æ–≤—ã–µ —Ü–µ–ø–∏ –∏ —Ä–µ–∑–æ–Ω–∞–Ω—Å–Ω—ã–µ —Ñ–∏–ª—å—Ç—Ä—ã

–ö–∞–∫ –∏–∑ –º–∏–ª–ª–∏–æ–Ω–∞ —Ä–∞–¥–∏–æ—Å—Ç–∞–Ω—Ü–∏–π –≤—ã–±—Ä–∞—Ç—å –æ–¥–Ω—É –∞–ø–ø–∞—Ä–∞—Ç–Ω—ã–º –ø—É—Ç–µ–º (–¥–æ –ø–æ—è–≤–ª–µ–Ω–∏—è —Ü–∏—Ñ—Ä–æ–≤—ã—Ö –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä–æ–≤)? –î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Ä–µ–∞–∫—Ç–∏–≤–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã, —Å–æ–ø—Ä–æ—Ç–∏–≤–ª–µ–Ω–∏–µ –∫–æ—Ç–æ—Ä—ã—Ö –∑–∞–≤–∏—Å–∏—Ç –æ—Ç —á–∞—Å—Ç–æ—Ç—ã:
1.  **–ö–æ–Ω–¥–µ–Ω—Å–∞—Ç–æ—Ä (–ï–º–∫–æ—Å—Ç—å, C):** –†–µ–∞–∫—Ç–∏–≤–Ω–æ–µ —Å–æ–ø—Ä–æ—Ç–∏–≤–ª–µ–Ω–∏–µ `X_C = 1 / (2*œÄ * f * C)`. –ü—Ä–∏ —É–≤–µ–ª–∏—á–µ–Ω–∏–∏ —á–∞—Å—Ç–æ—Ç—ã —Å–æ–ø—Ä–æ—Ç–∏–≤–ª–µ–Ω–∏–µ –ø–∞–¥–∞–µ—Ç (–ø—Ä–æ–ø—É—Å–∫–∞–µ—Ç –≤—ã—Å–æ–∫–∏–µ —á–∞—Å—Ç–æ—Ç—ã).
2.  **–ö–∞—Ç—É—à–∫–∞ (–ò–Ω–¥—É–∫—Ç–∏–≤–Ω–æ—Å—Ç—å, L):** –†–µ–∞–∫—Ç–∏–≤–Ω–æ–µ —Å–æ–ø—Ä–æ—Ç–∏–≤–ª–µ–Ω–∏–µ `X_L = 2*œÄ * f * L`. –ü—Ä–∏ —É–≤–µ–ª–∏—á–µ–Ω–∏–∏ —á–∞—Å—Ç–æ—Ç—ã —Å–æ–ø—Ä–æ—Ç–∏–≤–ª–µ–Ω–∏–µ —Ä–∞—Å—Ç–µ—Ç (–ø—Ä–æ–ø—É—Å–∫–∞–µ—Ç –Ω–∏–∑–∫–∏–µ —á–∞—Å—Ç–æ—Ç—ã).

–ï—Å–ª–∏ —Å–æ–µ–¥–∏–Ω–∏—Ç—å –∏—Ö –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω–æ –≤ **LC-–∫–æ–Ω—Ç—É—Ä**, –Ω–∞—Å—Ç—É–ø–∏—Ç –º–æ–º–µ–Ω—Ç, –∫–æ–≥–¥–∞ –∏—Ö —Ä–µ–∞–∫—Ç–∏–≤–Ω—ã–µ —Ç–æ–∫–∏ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ —É—Ä–∞–≤–Ω—è—é—Ç—Å—è –∏ –Ω–∞—á–Ω—É—Ç —Ü–∏—Ä–∫—É–ª–∏—Ä–æ–≤–∞—Ç—å –≤–Ω—É—Ç—Ä–∏ –∫–æ–Ω—Ç—É—Ä–∞ –±–µ–∑ –ø–æ—Ç–µ—Ä—å. –≠—Ç–æ —è–≤–ª–µ–Ω–∏–µ —Ä–µ–∑–æ–Ω–∞–Ω—Å–∞.
–ß–∞—Å—Ç–æ—Ç–∞ —Ä–µ–∑–æ–Ω–∞–Ω—Å–∞ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –ø–æ –∑–Ω–∞–º–µ–Ω–∏—Ç–æ–π **—Ñ–æ—Ä–º—É–ª–µ –¢–æ–º—Å–æ–Ω–∞**:
`f_res = 1 / (2*œÄ * sqrt(L * C))`
–ù–∞ —ç—Ç–æ–π —É–∑–∫–æ–π —á–∞—Å—Ç–æ—Ç–µ –∫–æ–Ω—Ç—É—Ä –æ–±—Ä–µ—Ç–∞–µ—Ç –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ–µ —Å–æ–ø—Ä–æ—Ç–∏–≤–ª–µ–Ω–∏–µ (–≤ –∏–¥–µ–∞–ª–µ) –∏ –≤—ã–¥–µ–ª—è–µ—Ç –ø–æ–ª–µ–∑–Ω—ã–π —Å–∏–≥–Ω–∞–ª –∏–∑ —à—É–º–∞. –°–ø–æ—Å–æ–±–Ω–æ—Å—Ç—å –∫–æ–Ω—Ç—É—Ä–∞ –≤—ã–¥–µ–ª—è—Ç—å —É–∑–∫—É—é –ø–æ–ª–æ—Å—É –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è **–¥–æ–±—Ä–æ—Ç–Ω–æ—Å—Ç—å—é (Q-—Ñ–∞–∫—Ç–æ—Ä)**.

# –ì–ª–∞–≤–∞ II: –ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞ —Å—É–ø–µ—Ä–≥–µ—Ç–µ—Ä–æ–¥–∏–Ω–∞ (–ü–µ—Ä–µ–Ω–æ—Å —Å–ø–µ–∫—Ç—Ä–∞)

–°–æ–∑–¥–∞—Ç—å –∏–¥–µ–∞–ª—å–Ω—ã–π, –ø–µ—Ä–µ—Å—Ç—Ä–∞–∏–≤–∞–µ–º—ã–π –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–º LC-—Ñ–∏–ª—å—Ç—Ä, –∫–æ—Ç–æ—Ä—ã–π —Å–æ—Ö—Ä–∞–Ω—è–ª –±—ã —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏ –≤ –æ–≥—Ä–æ–º–Ω–æ–º –¥–∏–∞–ø–∞–∑–æ–Ω–µ (–æ—Ç 0 –¥–æ 2000 –ú–ì—Ü), Î¨ºÎ¶¨—á–µ—Å–∫–∏ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ. –ò–Ω–∂–µ–Ω–µ—Ä—ã –ø—Ä–∏–º–µ–Ω–∏–ª–∏ –≥–µ–Ω–∏–∞–ª—å–Ω—ã–π —Ö–∞–∫: –º—ã –∑–∞—Ñ–∏–∫—Å–∏—Ä—É–µ–º –Ω–∞—à —Ñ–∏–ª—å—Ç—Ä –Ω–∞ –æ–¥–Ω–æ–π —É–¥–æ–±–Ω–æ–π –Ω–∏–∑–∫–æ–π —á–∞—Å—Ç–æ—Ç–µ, –∏ –±—É–¥–µ–º –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ "—Å–¥–≤–∏–≥–∞—Ç—å" –≤–µ—Å—å —ç—Ñ–∏—Ä –∫ —ç—Ç–æ–º—É —Ñ–∏–ª—å—Ç—Ä—É!

## 4. –§–∏–∑–∏–∫–∞ –Ω–µ–ª–∏–Ω–µ–π–Ω—ã—Ö —Ü–µ–ø–µ–π –∏ —Ä—è–¥ –¢–µ–π–ª–æ—Ä–∞

–°–¥–≤–∏–≥ —á–∞—Å—Ç–æ—Ç—ã —Ç—Ä–µ–±—É–µ—Ç –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–≥–æ **—É–º–Ω–æ–∂–µ–Ω–∏—è** —Å–∏–≥–Ω–∞–ª–∞ –∏–∑ –∞–Ω—Ç–µ–Ω–Ω—ã –Ω–∞ —Å–∏–≥–Ω–∞–ª –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–≥–æ –ª–æ–∫–∞–ª—å–Ω–æ–≥–æ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–∞ (–≥–µ—Ç–µ—Ä–æ–¥–∏–Ω–∞). –ü–∞—Å—Å–∏–≤–Ω—ã–µ —Ä–µ–∑–∏—Å—Ç–æ—Ä—ã, –∫–∞—Ç—É—à–∫–∏ –∏ –∫–æ–Ω–¥–µ–Ω—Å–∞—Ç–æ—Ä—ã —è–≤–ª—è—é—Ç—Å—è –ª–∏–Ω–µ–π–Ω—ã–º–∏ —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏, –æ–Ω–∏ —É–º–µ—é—Ç —Ç–æ–ª—å–∫–æ —Å–∫–ª–∞–¥—ã–≤–∞—Ç—å –Ω–∞–ø—Ä—è–∂–µ–Ω–∏—è ($U_{out} = c_1 U_1 + c_2 U_2$).

–£–º–Ω–æ–∂–µ–Ω–∏–µ –≤–æ–∑–º–æ–∂–Ω–æ —Ç–æ–ª—å–∫–æ –≤ **–Ω–µ–ª–∏–Ω–µ–π–Ω–æ–º —ç–ª–µ–º–µ–Ω—Ç–µ**, –Ω–∞–ø—Ä–∏–º–µ—Ä, –≤ –ø–æ–ª—É–ø—Ä–æ–≤–æ–¥–Ω–∏–∫–æ–≤–æ–º –¥–∏–æ–¥–µ. –í–æ–ª—å—Ç-–∞–º–ø–µ—Ä–Ω–∞—è —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞ (–í–ê–•) –¥–∏–æ–¥–∞ $I(U) = I_s e^{U / V_T}$ —è–≤–ª—è–µ—Ç—Å—è —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–π. 
–†–∞–∑–ª–æ–∂–∏–º –µ—ë –≤ —Ä—è–¥ –¢–µ–π–ª–æ—Ä–∞:
`I(U) = a_0 + a_1*U + a_2*U^2 + a_3*U^3 + ...`

–ï—Å–ª–∏ –ø–æ–¥–∞—Ç—å –Ω–∞ –¥–∏–æ–¥ —Å—É–º–º—É —Å–∏–≥–Ω–∞–ª–∞ –∞–Ω—Ç–µ–Ω–Ω—ã –∏ —Å–∏–≥–Ω–∞–ª–∞ –≥–µ—Ç–µ—Ä–æ–¥–∏–Ω–∞ $U = U_{rf} + U_{lo}$, —Ç–æ –≤ –∫–≤–∞–¥—Ä–∞—Ç–∏—á–Ω–æ–º —á–ª–µ–Ω–µ —Ä—è–¥–∞ –¢–µ–π–ª–æ—Ä–∞ $a_2 (U_{rf} + U_{lo})^2$ –ø–æ—è–≤–∏—Ç—Å—è –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–π —á–ª–µ–Ω-–ø–µ—Ä–µ–º–Ω–æ–∂–∏—Ç–µ–ª—å: $2 * a_2 * U_{rf} * U_{lo}$. –ò–º–µ–Ω–Ω–æ —Ñ–∏–∑–∏–∫–∞ –ø–æ–ª—É–ø—Ä–æ–≤–æ–¥–Ω–∏–∫–æ–≤ —Ä–æ–∂–¥–∞–µ—Ç —á–∏—Å—Ç–æ–µ —É–º–Ω–æ–∂–µ–Ω–∏–µ!

## 5. –ì–µ—Ç–µ—Ä–æ–¥–∏–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ –∏ Ï§ëÍ∞Ñ —á–∞—Å—Ç–æ—Ç–∞ (IF)

–ò—Ç–∞–∫, —É–º–Ω–æ–∂–∞–µ–º —Å–∏–≥–Ω–∞–ª –≤–µ—â–∞–Ω–∏—è –Ω–∞ —Å–∏–≥–Ω–∞–ª –ª–æ–∫–∞–ª—å–Ω–æ–≥–æ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–∞:
`U_mix(t) = cos(2*œÄ * f_rf * t) * cos(2*œÄ * f_lo * t)`

–ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∞—è —Ç—Ä–∏–≥–æ–Ω–æ–º–µ—Ç—Ä–∏—è –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏—è –∫–æ—Å–∏–Ω—É—Å–æ–≤ –æ—Ç–≤–µ—á–∞–µ—Ç –º–≥–Ω–æ–≤–µ–Ω–Ω–æ:
`cos(A)*cos(B) = 0.5*[cos(A - B) + cos(A + B)]`
`U_mix(t) = 0.5 * cos(2*œÄ * [f_rf - f_lo]*t)   +   0.5 * cos(2*œÄ * [f_rf + f_lo]*t)`

–ò–∑ —Å–º–µ—Å–∏—Ç–µ–ª—è –≤—ã–ª–µ—Ç–∞—é—Ç –¥–≤–µ –Ω–æ–≤—ã–µ –≤–æ–ª–Ω—ã:
1.  **–°—É–º–º–∞—Ä–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞:** `f_rf + f_lo`. –ß–∞—Å—Ç–æ—Ç–∞ –µ—â–µ –≤—ã—à–µ, —á–µ–º –±—ã–ª–∞. –ì–∞—Å–∏—Ç—Å—è –ø—Ä–æ—Å—Ç–µ–π—à–∏–º –∫–æ–Ω–¥–µ–Ω—Å–∞—Ç–æ—Ä–æ–º –Ω–∞ –≤—ã—Ö–æ–¥–µ.
2.  **–†–∞–∑–Ω–æ—Å—Ç–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞:** `f_rf - f_lo`. –≠—Ç–æ **–ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞ (IF, Intermediate Frequency)**. –ú—ã –∏–¥–µ–∞–ª—å–Ω–æ –ø–µ—Ä–µ–Ω–µ—Å–ª–∏ —Å–∏–≥–Ω–∞–ª –≤–Ω–∏–∑ —Å–ø–µ–∫—Ç—Ä–∞ –±–µ–∑ –ø–æ—Ç–µ—Ä–∏ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏!

**–ü—Ä–æ–±–ª–µ–º–∞ –∑–µ—Ä–∫–∞–ª—å–Ω–æ–≥–æ –∫–∞–Ω–∞–ª–∞:** –ï—Å–ª–∏ –º—ã —Å–ª—É—à–∞–µ–º —Ä–∞–¥–∏–æ –Ω–∞ —á–∞—Å—Ç–æ—Ç–µ 100 –ú–ì—Ü —Å –≥–µ—Ç–µ—Ä–æ–¥–∏–Ω–æ–º 90 –ú–ì—Ü, —Ç–æ IF —Ä–∞–≤–Ω–∞ `100 - 90 = 10 –ú–ì—Ü`. –ù–æ –µ—Å–ª–∏ –≤ —ç—Ñ–∏—Ä–µ –µ—Å—Ç—å –º–æ—â–Ω–∞—è —Å—Ç–∞–Ω—Ü–∏—è –Ω–∞ 80 –ú–ì—Ü, —Ç–æ –µ—ë —Å–∏–≥–Ω–∞–ª —Ç–∞–∫–∂–µ —É–º–Ω–æ–∂–∏—Ç—Å—è –Ω–∞ –≥–µ—Ç–µ—Ä–æ–¥–∏–Ω: `|80 - 90| = 10 –ú–ì—Ü`. –û–±–µ —Å—Ç–∞–Ω—Ü–∏–∏ —É–ø–∞–¥—É—Ç –≤ –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ IF-–∫–∞–Ω–∞–ª –∏ —É–Ω–∏—á—Ç–æ–∂–∞—Ç –¥—Ä—É–≥ –¥—Ä—É–≥–∞! –ü–æ—ç—Ç–æ–º—É –ø–µ—Ä–µ–¥ —Å–º–µ—Å–∏—Ç–µ–ª–µ–º —Å—Ç–∞–≤—è—Ç –≥—Ä—É–±—ã–π –∞–Ω–∞–ª–æ–≥–æ–≤—ã–π "–ø—Ä–µ—Å–µ–ª–µ–∫—Ç–æ—Ä", –æ—Ç—Å–µ–∫–∞—é—â–∏–π –∑–µ—Ä–∫–∞–ª—å–Ω—ã–µ —á–∞—Å—Ç–æ—Ç—ã –≤ –∞–Ω—Ç–µ–Ω–Ω–µ.

–¢–∞–∫ —É—Å—Ç—Ä–æ–µ–Ω —Ç—é–Ω–µ—Ä **R820T2** –Ω–∞ –ø–ª–∞—Ç–µ RTL-SDR: –æ–Ω —É—Å–∏–ª–∏–≤–∞–µ—Ç —Å–∏–≥–Ω–∞–ª –Ω–∞ –≤—Ö–æ–¥–µ, —É–º–Ω–æ–∂–∞–µ—Ç –µ–≥–æ –Ω–∞ –≥–µ—Ç–µ—Ä–æ–¥–∏–Ω, —Å–ø—É—Å–∫–∞—è –¥–æ –±–∞–∑–æ–≤–æ–π IF (3.57 –ú–ì—Ü) –∏ –æ—Ç—Ä–µ–∑–∞–µ—Ç –≤–µ—Å—å –í–ß-–º—É—Å–æ—Ä –¥–æ –ø–µ—Ä–µ–¥–∞—á–∏ –Ω–∞ —á–∏–ø –ê–¶–ü.

# –ì–ª–∞–≤–∞ III: –¢–µ–æ—Ä–∏—è –¥–∏—Å–∫—Ä–µ—Ç–∏–∑–∞—Ü–∏–∏ (–ú–æ—Å—Ç –≤ —Ü–∏—Ñ—Ä—É)

## 6. –ö–≤–∞–Ω—Ç–æ–≤–∞–Ω–∏–µ –∏ –ê–¶–ü RTL2832U

–ß–∏–ø RTL2832U –≤—Å—Ç—Ä–µ—á–∞–µ—Ç –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω—É—é —á–∞—Å—Ç–æ—Ç—É 3.57 –ú–ì—Ü –∏ –∏–∑–º–µ—Ä—è–µ—Ç –µ—ë –≤–æ–ª—å—Ç–∞–∂ 28.8 –º–∏–ª–ª–∏–æ–Ω–∞ —Ä–∞–∑ –≤ —Å–µ–∫—É–Ω–¥—É (MSPS) —Å —Ä–∞–∑—Ä–µ—à–µ–Ω–∏–µ–º 8 –±–∏—Ç. 

8 –±–∏—Ç –æ–∑–Ω–∞—á–∞–µ—Ç 256 —É—Ä–æ–≤–Ω–µ–π –Ω–∞–ø—Ä—è–∂–µ–Ω–∏—è (0..255). –®–∞–≥ –∫–≤–∞–Ω—Ç–æ–≤–∞–Ω–∏—è —Ä–æ–∂–¥–∞–µ—Ç –æ—à–∏–±–∫—É –æ–∫—Ä—É–≥–ª–µ–Ω–∏—è ‚Äî **—à—É–º –∫–≤–∞–Ω—Ç–æ–≤–∞–Ω–∏—è**. –§–∏–∑–∏—á–µ—Å–∫–∏–π –ø—Ä–µ–¥–µ–ª –û—Ç–Ω–æ—à–µ–Ω–∏—è –°–∏–≥–Ω–∞–ª/–®—É–º (SNR) –¥–ª—è –∏–¥–µ–∞–ª—å–Ω–æ–≥–æ N-–±–∏—Ç–Ω–æ–≥–æ –ê–¶–ü –∑–∞–¥–∞–µ—Ç—Å—è —Ñ–æ—Ä–º—É–ª–æ–π:
`SNR ‚âà 6.02*N + 1.76 [–≤ –î–µ—Ü–∏–±–µ–ª–∞—Ö]`
–î–ª—è –Ω–∞—à–µ–≥–æ RTL-SDR —ç—Ç–æ `6.02 * 8 + 1.76 ‚âà 50 –¥–ë`. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —Å–∞–º—ã–π –º–æ—â–Ω—ã–π —Å–∏–≥–Ω–∞–ª –º–æ–∂–µ—Ç –ø—Ä–µ–≤—ã—à–∞—Ç—å —Ç–µ–ø–ª–æ–≤–æ–π —à—É–º —á–∏–ø–∞ –≤—Å–µ–≥–æ –≤ 316 —Ä–∞–∑ (–ø–æ –∞–º–ø–ª–∏—Ç—É–¥–µ). (–î–ª—è —Å—Ä–∞–≤–Ω–µ–Ω–∏—è: 16-–±–∏—Ç–Ω—ã–µ Audio CD –∏–º–µ—é—Ç SNR ~96 –¥–ë).

## 7. –¢–µ–æ—Ä–µ–º–∞ –ö–æ—Ç–µ–ª—å–Ω–∏–∫–æ–≤–∞ (–ù–∞–π–∫–≤–∏—Å—Ç–∞-–®–µ–Ω–Ω–æ–Ω–∞)

–ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ –æ—Ü–∏—Ñ—Ä–æ–≤–∫–∞ ‚Äî —ç—Ç–æ —É–º–Ω–æ–∂–µ–Ω–∏–µ –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–≥–æ —Å–∏–≥–Ω–∞–ª–∞ `x(t)` –Ω–∞ –ø–µ—Ä–∏–æ–¥–∏—á–µ—Å–∫—É—é –≥—Ä–µ–±–µ–Ω–∫—É –¥–µ–ª—å—Ç–∞-—Ñ—É–Ω–∫—Ü–∏–π –î–∏—Ä–∞–∫–∞ (–®–∏-—Ñ—É–Ω–∫—Ü–∏—é). –í —á–∞—Å—Ç–æ—Ç–Ω–æ–π –æ–±–ª–∞—Å—Ç–∏ (–ø–æ —Å–≤–æ–π—Å—Ç–≤–∞–º —Å–≤–µ—Ä—Ç–∫–∏) —ç—Ç–æ –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ **–±–µ—Å–∫–æ–Ω–µ—á–Ω–æ–º—É —Ä–∞–∑–º–Ω–æ–∂–µ–Ω–∏—é —Å–ø–µ–∫—Ç—Ä–∞** $X(f)$ —Å —à–∞–≥–æ–º, —Ä–∞–≤–Ω—ã–º —á–∞—Å—Ç–æ—Ç–µ –¥–∏—Å–∫—Ä–µ—Ç–∏–∑–∞—Ü–∏–∏ $F_s$.

**–§—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–∞—è –¢–µ–æ—Ä–µ–º–∞ –ö–æ—Ç–µ–ª—å–Ω–∏–∫–æ–≤–∞:**
–ß—Ç–æ–±—ã —ç—Ç–∏ —Ä–∞–∑–º–Ω–æ–∂–µ–Ω–Ω—ã–µ —Å–ø–µ–∫—Ç—Ä–∞–ª—å–Ω—ã–µ –∫–æ–ø–∏–∏ –Ω–µ –Ω–∞–ª–æ–∂–∏–ª–∏—Å—å –¥—Ä—É–≥ –Ω–∞ –¥—Ä—É–≥–∞, –º–∞–∫—Å–∏–º–∞–ª—å–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞ –≤ —Å–∏–≥–Ω–∞–ª–µ $F_{max}$ –Ω–µ –¥–æ–ª–∂–Ω–∞ –ø—Ä–µ–≤—ã—à–∞—Ç—å –ø–æ–ª–æ–≤–∏–Ω—ã —á–∞—Å—Ç–æ—Ç—ã —Å—Ç—Ä–æ–±–∏—Ä–æ–≤–∞–Ω–∏—è:
`F_s > 2 * F_max`

–ï—Å–ª–∏ $F_{max}$ –ø—Ä–µ–≤—ã—à–∞–µ—Ç –ø—Ä–µ–¥–µ–ª –ù–∞–π–∫–≤–∏—Å—Ç–∞ ($F_s/2$), "—Ñ–∞–Ω—Ç–æ–º–Ω—ã–µ" –≤—ã—Å–æ–∫–æ—á–∞—Å—Ç–æ—Ç–Ω—ã–µ —Ö–≤–æ—Å—Ç—ã –æ—Ç—Ä–∞–∑—è—Ç—Å—è –æ—Ç –∫—Ä–∞—è —Å–ø–µ–∫—Ç—Ä–∞ –æ–±—Ä–∞—Ç–Ω–æ –∫ –Ω—É–ª—é, —Ä–∞–∑—Ä—É—à–∞—è –ø–æ–ª–µ–∑–Ω—ã–π —Å–∏–≥–Ω–∞–ª —Ç—è–∂–µ–ª—ã–º —Å–∫—Ä–µ–∂–µ—Ç–æ–º. –≠—Ç–æ —è–≤–ª–µ–Ω–∏–µ ‚Äî **–ê–ª–∏–∞—Å–∏–Ω–≥ (Aliasing)**. –ò–º–µ–Ω–Ω–æ –ø–æ—ç—Ç–æ–º—É –ø–µ—Ä–µ–¥ –∫–∞–∂–¥—ã–º –ê–¶–ü –∂–µ–ª–µ–∑–Ω–æ —Å—Ç–∞–≤—è—Ç –∞–Ω–∞–ª–æ–≥–æ–≤—ã–π –§–ù–ß-—Ñ–∏–ª—å—Ç—Ä.

## 8. –ö–≤–∞–¥—Ä–∞—Ç—É—Ä–Ω–∞—è –¥–µ–º–æ–¥—É–ª—è—Ü–∏—è (I/Q –¥–∞–Ω–Ω—ã–µ) –∏ –§–æ—Ä–º—É–ª–∞ –≠–π–ª–µ—Ä–∞

–°—É—â–µ—Å—Ç–≤—É–µ—Ç —Å–µ—Ä—å–µ–∑–Ω–∞—è –ø—Ä–æ–±–ª–µ–º–∞ –æ–¥–Ω–æ–º–µ—Ä–Ω–æ–π –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω–æ–π –æ—Ü–∏—Ñ—Ä–æ–≤–∫–∏. 
–°–æ–≥–ª–∞—Å–Ω–æ —Ç–æ–∂–¥–µ—Å—Ç–≤—É –≠–π–ª–µ—Ä–∞: 
`cos(œât) = 0.5 * (e^(j*œât) + e^(-j*œât))`
–í–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–π –∫–æ—Å–∏–Ω—É—Å —è–≤–ª—è–µ—Ç—Å—è —Å—É–º–º–æ–π –¥–≤—É—Ö –∫–æ–º–ø–ª–µ–∫—Å–Ω—ã—Ö —ç–∫—Å–ø–æ–Ω–µ–Ω—Ç (–≤—Ä–∞—â–∞—é—â–∏—Ö—Å—è –≤–µ–∫—Ç–æ—Ä–æ–≤), –∫—Ä—É—Ç—è—â–∏—Ö—Å—è –≤ –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω—ã–µ —Å—Ç–æ—Ä–æ–Ω—ã. –û–¥–Ω–æ–º–µ—Ä–Ω–∞—è –æ—Ü–∏—Ñ—Ä–æ–≤–∫–∞ –Ω–µ —Ä–∞–∑–ª–∏—á–∞–µ—Ç, –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –ª–∏ —á–∞—Å—Ç–æ—Ç–∞ –Ω–∞ $+100$ –∫–ì—Ü –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –Ω—É–ª—è, –∏–ª–∏ –Ω–∞ $-100$ –∫–ì—Ü. –ò—Ö —Å–ø–µ–∫—Ç—Ä—ã —Å—Ö–ª–æ–ø–Ω—É—Ç—Å—è, —É–Ω–∏—á—Ç–æ–∂–∏–≤ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–æ–Ω–Ω—É—é –∞—Å–∏–º–º–µ—Ç—Ä–∏—é.

**–ö–≤–∞–¥—Ä–∞—Ç—É—Ä–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ (–°–∏–Ω—Ñ–∞–∑–Ω–æ-–ö–≤–∞–¥—Ä–∞—Ç—É—Ä–Ω—ã–π –î–µ—Ç–µ–∫—Ç–æ—Ä):**
–ú—ã –ø—Ä–æ–ø—É—Å–∫–∞–µ–º —Å–∏–≥–Ω–∞–ª —á–µ—Ä–µ–∑ *–¥–≤–∞* –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω—ã—Ö —Å–º–µ—Å–∏—Ç–µ–ª—è. –û–ø–æ—Ä–Ω—ã–µ —Å–∏–≥–Ω–∞–ª—ã –≥–µ—Ç–µ—Ä–æ–¥–∏–Ω–∞ –¥–ª—è –Ω–∏—Ö —Ñ–∏–∑–∏—á–µ—Å–∫–∏ —Å–¥–≤–∏–Ω—É—Ç—ã —Ç–æ—á–Ω–æ –Ω–∞ 90¬∞ (œÄ/2).
1. **–ö–∞–Ω–∞–ª I (–°–∏–Ω—Ñ–∞–∑–Ω—ã–π):** –£–º–Ω–æ–∂–µ–Ω–∏–µ –Ω–∞ `cos(œâ_c * t)`.
2. **–ö–∞–Ω–∞–ª Q (–ö–≤–∞–¥—Ä–∞—Ç—É—Ä–Ω—ã–π):** –£–º–Ω–æ–∂–µ–Ω–∏–µ –Ω–∞ `-sin(œâ_c * t)`.

–ü–æ—Å–ª–µ —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏–∏ –ê–¶–ü –≤—ã–¥–∞–µ—Ç –ø–∞—Ä—ã –±–∞–π—Ç: `[I, Q, I, Q...]`. 
–ú—ã –ø—Ä–æ–≥—Ä–∞–º–º–Ω–æ —Å–æ–±–∏—Ä–∞–µ–º –∏–∑ –Ω–∏—Ö –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–µ —á–∏—Å–ª–æ: `Z = I + j*Q`.
–≠—Ç–æ **–ê–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–∏–π —Å–∏–≥–Ω–∞–ª**. –ü–æ —Ñ–æ—Ä–º—É–ª–µ –≠–π–ª–µ—Ä–∞ `e^(j*Œ∏) = cos(Œ∏) + j*sin(Œ∏)` –º—ã –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–∏–ª–∏ –¥–≤—É–º–µ—Ä–Ω—ã–π –≤–µ–∫—Ç–æ—Ä, –≤—Ä–∞—â–∞—é—â–∏–π—Å—è –Ω–∞ –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏. –ú–æ–¥—É–ª—å –≤–µ–∫—Ç–æ—Ä–∞ ‚Äî —ç—Ç–æ –∞–º–ø–ª–∏—Ç—É–¥–∞, —É–≥–æ–ª –≤–µ–∫—Ç–æ—Ä–∞ ‚Äî –ø–æ–ª–Ω–∞—è —Ñ–∞–∑–∞, –∞ –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏–µ –≤—Ä–∞—â–µ–Ω–∏—è (–ø–æ –∏–ª–∏ –ø—Ä–æ—Ç–∏–≤ —á–∞—Å–æ–≤–æ–π —Å—Ç—Ä–µ–ª–∫–∏) –≤–æ—Å—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç –∑–Ω–∞–Ω–∏–µ –æ —Ç–æ–º, –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–∞—è –ø–µ—Ä–µ–¥ –Ω–∞–º–∏ —á–∞—Å—Ç–æ—Ç–∞ –∏–ª–∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–∞—è!

–° —ç—Ç–æ–≥–æ –º–æ–º–µ–Ω—Ç–∞ `Z[n]` ‚Äî —ç—Ç–æ –Ω–∞—à $0$ –ì—Ü Baseband —Å–∏–≥–Ω–∞–ª —á–∞—Å—Ç–æ—Ç–æ–π 2.4 MSPS, –ø–µ—Ä–µ–¥–∞–Ω–Ω—ã–π –≤ Python.

# –ì–ª–∞–≤–∞ IV: –¶–∏—Ñ—Ä–æ–≤–∞—è –æ–±—Ä–∞–±–æ—Ç–∫–∞ —Å–∏–≥–Ω–∞–ª–æ–≤ (–¶–û–°)

## 9. –¶–∏—Ñ—Ä–æ–≤–∞—è —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏—è –∏ –§—É–Ω–∫—Ü–∏—è sinc(x)

–ü—Ä–∏–Ω—è—Ç–∞—è I/Q —à–∏—Ä–æ—Ç–Ω–∞—è –ø–æ–ª–æ—Å–∞ (–¥–æ–ø—É—Å—Ç–∏–º 2.4 –ú–ì—Ü) —Å–æ–¥–µ—Ä–∂–∏—Ç —Å–æ—Ç–Ω–∏ —Å—Ç–∞–Ω—Ü–∏–π. –ù–∞—à –ø–æ–ª–µ–∑–Ω—ã–π –∑–≤—É–∫ FM-—Å—Ç–∞–Ω—Ü–∏–∏ –∑–∞–Ω–∏–º–∞–µ—Ç –≤—Å–µ–≥–æ 150 –∫–ì—Ü. –û—Å—Ç–∞–ª—å–Ω–æ–µ ‚Äî —à—É–º. –ú—ã –æ–±—è–∑–∞–Ω—ã –æ—á–∏—Å—Ç–∏—Ç—å –º–∞—Å—Å–∏–≤ –≤ —Ü–∏—Ñ—Ä–æ–≤–æ–º –≤–∏–¥–µ.
–°—É—â–µ—Å—Ç–≤—É—é—Ç –¥–≤–∞ –∫–ª–∞—Å—Å–∞ —Ü–∏—Ñ—Ä–æ–≤—ã—Ö —Ñ–∏–ª—å—Ç—Ä–æ–≤:
*   **–ë–ò–• (IIR) —Ñ–∏–ª—å—Ç—Ä—ã:** –û—Å–Ω–æ–≤–∞–Ω—ã –Ω–∞ —Ä–∞–∑–Ω–æ—Å—Ç–Ω—ã—Ö —É—Ä–∞–≤–Ω–µ–Ω–∏—è—Ö —Å –æ–±—Ä–∞—Ç–Ω–æ–π —Å–≤—è–∑—å—é: `y[n] = x[n] + a*y[n-1]`. –ê–Ω–∞–ª–æ–≥–∏ —Ñ–∏–∑–∏—á–µ—Å–∫–∏—Ö LC-–∫–æ–Ω—Ç—É—Ä–æ–≤. –≠–∫–æ–Ω–æ–º—è—Ç –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä, –Ω–æ –∫–æ—Ä—ë–∂–∞—Ç —Ñ–∞–∑—É —Å–∏–≥–Ω–∞–ª–∞ (–≤–Ω–æ—Å—è—Ç –∑–∞–¥–µ—Ä–∂–∫–∏ –¥–ª—è —Ä–∞–∑–Ω—ã—Ö —á–∞—Å—Ç–æ—Ç).
*   **–ö–ò–• (FIR) —Ñ–∏–ª—å—Ç—Ä—ã:** –û—Å–Ω–æ–≤–∞–Ω—ã –Ω–∞ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–π –°–≤–µ—Ä—Ç–∫–µ: `y[n] = Œ£ h[k] * x[n-k]`. –ó–∞–¥–µ—Ä–∂–∫–∞ –¥–ª—è –≤—Å–µ—Ö —á–∞—Å—Ç–æ—Ç –æ–¥–∏–Ω–∞–∫–æ–≤–∞ (–ª–∏–Ω–µ–π–Ω–∞—è –§–ß–•), –∏–¥–µ–∞–ª—å–Ω—ã –¥–ª—è SDR. 

–°–æ–≥–ª–∞—Å–Ω–æ –æ–±—Ä–∞—Ç–Ω–æ–º—É –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—é –§—É—Ä—å–µ, –∏–¥–µ–∞–ª—å–Ω—ã–π –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω—ã–π —Ñ–∏–ª—å—Ç—Ä –Ω–∏–∂–Ω–∏—Ö —á–∞—Å—Ç–æ—Ç (–ü-–æ–±—Ä–∞–∑–Ω—ã–π –§–ù–ß) –≤–æ –≤—Ä–µ–º–µ–Ω–Ω–æ–π –æ–±–ª–∞—Å—Ç–∏ –æ–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –∏–º–ø—É–ª—å—Å–Ω–æ–π —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–æ–π –≤ –≤–∏–¥–µ —Ñ—É–Ω–∫—Ü–∏–∏ **`sinc(x) = sin(x)/x`**. –û–Ω–∞ –±–µ—Å–∫–æ–Ω–µ—á–Ω–∞, –ø–æ—ç—Ç–æ–º—É –º—ã "—É—Å–µ–∫–∞–µ–º" –æ–∫–Ω–æ –∫–æ–Ω–µ—á–Ω—ã–º —á–∏—Å–ª–æ–º –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–æ–≤ `h[k]`, —á—Ç–æ —Ä–æ–∂–¥–∞–µ—Ç –ø—É–ª—å—Å–∞—Ü–∏–∏ (–≠—Ñ—Ñ–µ–∫—Ç –ì–∏–±–±—Å–∞).

## 10. –î–µ—Ü–∏–º–∞—Ü–∏—è –∏ Box-car Integrator

–î–µ—Ü–∏–º–∞—Ü–∏—è ‚Äî —ç—Ç–æ –ø—Ä–æ—Ä–µ–∂–∏–≤–∞–Ω–∏–µ –º–∞—Å—Å–∏–≤–∞ –¥–ª—è –ø–æ–Ω–∏–∂–µ–Ω–∏—è —á–∞—Å—Ç–æ—Ç—ã –¥–∏—Å–∫—Ä–µ—Ç–∏–∑–∞—Ü–∏–∏ ($F_s \rightarrow F_s / D$).
–ü–æ –∑–∞–∫–æ–Ω—É –ù–∞–π–∫–≤–∏—Å—Ç–∞, –ø–µ—Ä–µ–¥ —Å–±—Ä–æ—Å–æ–º —Å—ç–º–ø–ª–æ–≤ –º—ã –æ–±—è–∑–∞–Ω—ã –ø—Ä–∏–º–µ–Ω–∏—Ç—å –∂–µ—Å—Ç–æ—á–∞–π—à–∏–π –§–ù–ß, —á—Ç–æ–±—ã –∏–∑–±–µ–∂–∞—Ç—å –∞–ª–∏–∞—Å–∏–Ω–≥–∞.

–í Python –¥–ª—è —Å–≤–µ—Ä—Ö–±—ã—Å—Ç—Ä–æ–π –¥–µ—Ü–∏–º–∞—Ü–∏–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –≤ $D=5$ —Ä–∞–∑) –º—ã –ø—Ä–∏–º–µ–Ω—è–µ–º —Ñ–∏–ª—å—Ç—Ä –°–∫–æ–ª—å–∑—è—â–µ–≥–æ –°—Ä–µ–¥–Ω–µ–≥–æ (**Box-car integrator**).
–£—Å—Ä–µ–¥–Ω–µ–Ω–∏–µ $D$ —Å–æ—Å–µ–¥–Ω–∏—Ö –æ—Ç—Å—á–µ—Ç–æ–≤ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ –ö–ò–•-—Å–≤–µ—Ä—Ç–∫–µ —Å –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω—ã–º –æ–∫–Ω–æ–º –µ–¥–∏–Ω–∏—Ü: `h = [1/D, 1/D, ... 1/D]`. 
–ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –§—É—Ä—å–µ –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω–∏–∫–∞ –¥–∞–µ—Ç –ê–ß–• –≤ –≤–∏–¥–µ –º–æ–¥—É–ª—è `|sin(œÄ f D) / (D sin(œÄ f))|`. –ü–æ—Ç—Ä—è—Å–∞—é—â–µ–µ —Å–≤–æ–π—Å—Ç–≤–æ —ç—Ç–æ–π –ê–ß–• –≤ —Ç–æ–º, —á—Ç–æ –µ—ë –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–µ –Ω—É–ª–∏ (—Ç–æ—á–∫–∏ –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ–≥–æ –ø–æ–¥–∞–≤–ª–µ–Ω–∏—è) –Ω–∞—Ö–æ–¥—è—Ç—Å—è —Ä–æ–≤–Ω–æ –Ω–∞ –∫—Ä–∞—Ç–Ω–æ—Å—Ç—è—Ö –Ω–æ–≤–æ–π —á–∞—Å—Ç–æ—Ç—ã –≤—ã–±–æ—Ä–∫–∏ ‚Äî –∏–º–µ–Ω–Ω–æ —Ç–∞–º, –≥–¥–µ –ø—Ä—è—á–µ—Ç—Å—è –≥–ª–∞–≤–Ω–∞—è —ç–Ω–µ—Ä–≥–∏—è –∞–ª–∏–∞—Å–∏–Ω–≥–∞!

```python
# –ö–ò–• –∞–Ω—Ç–∏-–∞–ª–∏–∞—Å —Ñ–∏–ª—å—Ç—Ä + –î–µ—Ü–∏–º–∞—Ü–∏—è –≤ D=5 —Ä–∞–∑
n_decimated = len(i_channel) - (len(i_channel) % 5)

audio_baseband = [
    # –£—Å—Ä–µ–¥–Ω—è–µ–º –±–ª–æ–∫ –∏–∑ 5 –æ—Ç—Å—á–µ—Ç–æ–≤ (sinc(x) –∞–Ω—Ç–∏–∞–ª–∏–∞—Å–∏–Ω–≥) –∏ 
    # –ø—Ä—ã–≥–∞–µ–º –Ω–∞ 5 —à–∞–≥–æ–≤ –≤–ø–µ—Ä–µ–¥ (–¥–µ—Ü–∏–º–∞—Ü–∏—è)
    (i_channel[j] + i_channel[j+1] + i_channel[j+2] + i_channel[j+3] + i_channel[j+4]) / 5.0
    for j in range(0, n_decimated, 5)
]
```

# –ì–ª–∞–≤–∞ V: –¢–µ–æ—Ä–∏—è –º–æ–¥—É–ª—è—Ü–∏–∏ –∏ —Ü–∏—Ñ—Ä–æ–≤—ã–µ –¥–µ–º–æ–¥—É–ª—è—Ç–æ—Ä—ã

## 11. –ê–º–ø–ª–∏—Ç—É–¥–Ω–∞—è –º–æ–¥—É–ª—è—Ü–∏—è (AM)

**–§–∏–∑–∏–∫–∞ –ø–µ—Ä–µ–¥–∞—á–∏:**
–ó–≤—É–∫–æ–≤–æ–π —Å–∏–≥–Ω–∞–ª `m(t)` –º–µ–Ω—è–µ—Ç –∞–º–ø–ª–∏—Ç—É–¥—É –≤—ã—Å–æ–∫–æ—á–∞—Å—Ç–æ—Ç–Ω–æ–π –Ω–µ—Å—É—â–µ–π –≤–æ–ª–Ω—ã. –ò–Ω–¥–µ–∫—Å –≥–ª—É–±–∏–Ω—ã –º–æ–¥—É–ª—è—Ü–∏–∏ $M \le 1$.
`s_am(t) = [A_0 + m(t)] * cos(œâ_c * t)` 
–ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ —Ä–∞—Å–∫—Ä—ã—Ç–∏–µ —Å–∫–æ–±–æ–∫ –ø–æ—Ä–æ–∂–¥–∞–µ—Ç $A_0 \cos(œâ_c t) + 0.5 m(t) \cos((œâ_c - œâ_m)t) + 0.5 m(t) \cos((œâ_c + œâ_m)t)$. 
–ú—ã –≤–∏–¥–∏–º –±–µ—Å–ø–æ–ª–µ–∑–Ω—ã–π –ø–∏–∫ –Ω–µ—Å—É—â–µ–π $A_0$, –∑–∞–±–∏—Ä–∞—é—â–∏–π 66% —ç–ª–µ–∫—Ç—Ä–∏—á–µ—Å–∫–æ–π –º–æ—â–Ω–æ—Å—Ç–∏ –ø–µ—Ä–µ–¥–∞—Ç—á–∏–∫–∞, –∏ –¥–≤–µ –∑–µ—Ä–∫–∞–ª—å–Ω—ã–µ –±–æ–∫–æ–≤—ã–µ –ø–æ–ª–æ—Å—ã —Å –ø–æ–ª–µ–∑–Ω—ã–º –∑–≤—É–∫–æ–º $m(t)$.

**–¶–∏—Ñ—Ä–æ–≤–æ–π –¥–µ–º–æ–¥—É–ª—è—Ç–æ—Ä:**
–ö–æ–≥–¥–∞ RTL-SDR –∫–≤–∞–¥—Ä–∞—Ç—É—Ä–Ω–æ –ø–µ—Ä–µ–Ω–æ—Å–∏—Ç —Å–ø–µ–∫—Ç—Ä –≤ 0 –ì—Ü (Baseband), –Ω–µ—Å—É—â–∞—è —á–∞—Å—Ç–æ—Ç–∞ —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–º —Ç–æ–∫–æ–º DC ($0$ –ì—Ü).
–ü—Ä–∏ —ç—Ç–æ–º –∑–≤—É–∫–æ–≤–∞—è –æ–≥–∏–±–∞—é—â–∞—è `m(t)` –ø–æ–ª–Ω–æ—Å—Ç—å—é –∏ –∞–±—Å–æ–ª—é—Ç–Ω–æ –∑–∞–¥–∞–µ—Ç –≤–µ–∫—Ç–æ—Ä–Ω—É—é –¥–ª–∏–Ω—É –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–≥–æ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–∞ `Z = I + j*Q`. –£–≥–æ–ª –≤–µ–∫—Ç–æ—Ä–∞ –ø—Ä–∏ —á–∏—Å—Ç–æ–π –∞–º–ø–ª–∏—Ç—É–¥–Ω–æ–π –º–∞–Ω–∏–ø—É–ª—è—Ü–∏–∏ –Ω–µ –Ω–µ—Å–µ—Ç –Ω–∏–∫–∞–∫–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏. 

–ê–ª–≥–æ—Ä–∏—Ç–º –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ –∏–∑–≤–ª–µ—á–µ–Ω–∏–∏ –º–æ–¥—É–ª—è –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–≥–æ –≤–µ–∫—Ç–æ—Ä–∞ (–î–µ—Ç–µ–∫—Ç–æ—Ä –æ–≥–∏–±–∞—é—â–µ–π) —Å –ø–æ—Å–ª–µ–¥—É—é—â–∏–º —É–¥–∞–ª–µ–Ω–∏–µ–º —Å–º–µ—â–µ–Ω–∏—è $A_0$ —Å –ø–æ–º–æ—â—å—é –ø—Ä–æ—Å—Ç–µ–π—à–µ–≥–æ –æ–¥–Ω–æ–ø–æ–ª—é—Å–Ω–æ–≥–æ —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ–≥–æ –ë–ò–• –§–í–ß-—Ñ–∏–ª—å—Ç—Ä–∞.

```python
audio_env = []
for k in range(n_pairs):
    i = raw[k*2] - 127.5
    q = raw[k*2+1] - 127.5
    
    # –¢–µ–æ—Ä–µ–º–∞ –ü–∏—Ñ–∞–≥–æ—Ä–∞ –≤—ã—á–∏—Å–ª—è–µ—Ç –º–æ–¥—É–ª—å –≤–µ–∫—Ç–æ—Ä–∞
    magnitude = math.sqrt(i**2 + q**2)
    audio_env.append(magnitude)

# 1-–ø–æ–ª—é—Å–Ω—ã–π –ë–ò–• (IIR) —Ñ–∏–ª—å—Ç—Ä –≤—ã—Å–æ–∫–∏—Ö —á–∞—Å—Ç–æ—Ç (–£–¥–∞–ª–µ–Ω–∏–µ –Ω–µ—Å—É—â–µ–≥–æ —Å–∫–∞–ª—è—Ä–∞ DC)
# dc_accum –∏–Ω—Ç–µ–≥—Ä–∏—Ä—É–µ—Ç "—Ö–æ–ª–æ—Å—Ç—É—é" –º–æ—â–Ω–æ—Å—Ç—å, –º—ã –≤—ã—á–∏—Ç–∞–µ–º –µ—ë –∏–∑ –∑–≤—É–∫–∞
dc_accum = 0.0
alpha = 0.001
for idx in range(len(audio_env)):
    dc_accum += alpha * (audio_env[idx] - dc_accum)
    audio_env[idx] -= dc_accum
```

## 12. –û–¥–Ω–æ–ø–æ–ª–æ—Å–Ω–∞—è –º–æ–¥—É–ª—è—Ü–∏—è (SSB: USB / LSB)

**–§–∏–∑–∏–∫–∞ –ø–µ—Ä–µ–¥–∞—á–∏:**
SSB (Single Sideband) ‚Äî –≤–µ—Ä—à–∏–Ω–∞ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏ –∞–Ω–∞–ª–æ–≥–æ–≤–æ–≥–æ —ç—Ñ–∏—Ä–∞. –ß—Ç–æ–±—ã –Ω–µ –≥—Ä–µ—Ç—å –∞—Ç–º–æ—Å—Ñ–µ—Ä—É 66% –º–æ—â–Ω–æ—Å—Ç—å—é –Ω–µ—Å—É—â–µ–π –∏ –Ω–µ –¥—É–±–ª–∏—Ä–æ–≤–∞—Ç—å —Å–ø–µ–∫—Ç—Ä, –ø–µ—Ä–µ–¥–∞—Ç—á–∏–∫ SSB (–∏—Å–ø–æ–ª—å–∑—É—è –∫–≤–∞—Ä—Ü–µ–≤—ã–µ —Ñ–∏–ª—å—Ç—Ä—ã –∏–ª–∏ —Ñ–∞–∑–æ–≤—ã–π –º–µ—Ç–æ–¥) –ø–æ–ª–Ω–æ—Å—Ç—å—é –≤—ã—Ä–µ–∑–∞–µ—Ç –Ω–µ—Å—É—â—É—é –∏ –æ–¥–Ω—É –∏–∑ –±–æ–∫–æ–≤—ã—Ö –ø–æ–ª–æ—Å. 100% –º–æ—â–Ω–æ—Å—Ç–∏ –≤–∫–ª–∞–¥—ã–≤–∞–µ—Ç—Å—è –≤ 3 –∫–ì—Ü —É–∑–∫–æ–≥–æ –≥–æ–ª–æ—Å–∞. 
*   **USB:** –ì–æ–ª–æ—Å –ª–µ–∂–∏—Ç –≤ `f_c + 0...3 –∫–ì—Ü`.
*   **LSB:** –ì–æ–ª–æ—Å –∏–Ω–≤–µ—Ä—Ç–∏—Ä–æ–≤–∞–Ω –∏ –ª–µ–∂–∏—Ç –≤ `f_c - 0...3 –∫–ì—Ü`.

–ì–µ–Ω–µ—Ä–∞—Ü–∏—è SSB –≤ —Ü–∏—Ñ—Ä–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –≤ –Ω–∞—à–µ–º —Å–µ—Ä–≤–µ—Ä–µ-–º–æ–∫–µ—Ä–µ) —Ç—Ä–µ–±—É–µ—Ç –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –ê–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–æ–≥–æ —Å–∏–≥–Ω–∞–ª–∞ –∑–≤—É–∫–∞ `z_audio(t) = m(t) + j*H_m(t)`, –≥–¥–µ `H_m(t)` ‚Äî **–ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –ì–∏–ª—å–±–µ—Ä—Ç–∞**, –ø–æ–≤–æ—Ä–∞—á–∏–≤–∞—é—â–µ–µ –≤—Å–µ —á–∞—Å—Ç–æ—Ç–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã –∑–≤—É–∫–∞ –Ω–∞ —Ñ–∞–∑—É —Å—Ç—Ä–æ–≥–æ $\pi/2$. –ü–æ–¥—Å—Ç–∞–≤–ª—è—è —ç—Ç–æ –≤ —É–º–Ω–æ–∂–∏—Ç–µ–ª—å, –º—ã –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ –¥–µ—Å—Ç—Ä—É–∫—Ç–∏–≤–Ω–æ –ø–æ–¥–∞–≤–ª—è–µ–º –Ω–µ–∂–µ–ª–∞—Ç–µ–ª—å–Ω—É—é –±–æ–∫–æ–≤—É—é –ø–æ–ª–æ—Å—É –≤ –Ω–æ–ª—å.

**–ê–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–æ–µ —Å–ø–µ–∫—Ç—Ä–∞–ª—å–Ω–æ–µ —Å–≤–æ—Ä–∞—á–∏–≤–∞–Ω–∏–µ (–î–µ–º–æ–¥—É–ª—è—Ç–æ—Ä):**
–î–µ–º–æ–¥—É–ª—è—Ü–∏—è SSB –≤ Baseband'–µ –∏–∑—ã—Å–∫–∞–Ω–Ω–æ –∫—Ä–∞—Å–∏–≤–∞. 
–ï—Å–ª–∏ Baseband –æ—Ç—Ü–µ–Ω—Ç—Ä–æ–≤–∞–Ω –ø–æ —É–¥–∞–ª–µ–Ω–Ω–æ–π –Ω–µ—Å—É—â–µ–π `0 –ì—Ü`, —Ç–æ –∞–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–∏–π —Å–ø–µ–∫—Ç—Ä USB –ª–µ–∂–∏—Ç —Ü–µ–ª–∏–∫–æ–º –≤ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–π $f > 0$ –ø–æ–ª—É–ø–ª–æ—Å–∫–æ—Å—Ç–∏ –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–π —Å–∏—Å—Ç–µ–º—ã, –∞ LSB ‚Äî –≤ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–π $f < 0$.

–§–∏–∑–∏—á–µ—Å–∫–∏–π –¥–∏—Ñ—Ñ—É–∑–æ—Ä –¥–∏–Ω–∞–º–∏–∫–∞ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –æ–¥–Ω–æ–º–µ—Ä–Ω—ã–µ 1D –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–µ —á–∏—Å–ª–∞. –ü–æ –∑–∞–∫–æ–Ω–∞–º –≥–µ–æ–º–µ—Ç—Ä–∏–∏ –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–π –ø–ª–æ—Å–∫–æ—Å—Ç–∏, `Z + Z* = 2 * Real(Z)`. –ï—Å–ª–∏ –º—ã –ø—Ä–æ—Å—Ç–æ –æ—Ç–±—Ä–æ—Å–∏–º –ú–Ω–∏–º—É—é —á–∞—Å—Ç—å `Q` —É –ø—Ä–∏–Ω—è—Ç–æ–≥–æ —Å–∏–≥–Ω–∞–ª–∞ –∏ –æ—Å—Ç–∞–≤–∏–º –°–∏–Ω—Ñ–∞–∑–Ω—É—é `I`, –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–µ –∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ —á–∞—Å—Ç–æ—Ç—ã –≤—ã–Ω—É–∂–¥–µ–Ω—ã –±—É–¥—É—Ç –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ "—Å—Ö–ª–µ—Å—Ç–Ω—É—Ç—å—Å—è" –∏ –∑–µ—Ä–∫–∞–ª—å–Ω–æ —Å–ø—Ä–æ–µ—Ü–∏—Ä–æ–≤–∞—Ç—å—Å—è –¥—Ä—É–≥ –Ω–∞ –¥—Ä—É–≥–∞, —Ç–∞–∫ –∫–∞–∫ —Å–ø–µ–∫—Ç—Ä –≤–µ—â–µ—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ —Å–∏–≥–Ω–∞–ª–∞ –∂–µ—Å—Ç–∫–æ —Å–∏–º–º–µ—Ç—Ä–∏—á–µ–Ω.

```python
# –°–ø–µ–∫—Ç—Ä–∞–ª—å–Ω–æ–µ —Å—Ö–ª–æ–ø—ã–≤–∞–Ω–∏–µ I/Q -> 1D Audio
for k in range(n_pairs):

    # –≠—Ç–∞ –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–∞—è —Å—Ç—Ä–æ–∫–∞ –ú–ì–ù–û–í–ï–ù–ù–û –ø–µ—Ä–µ—Å—Ç—Ä–∞–∏–≤–∞–µ—Ç –≤–µ–∫—Ç–æ—Ä –≤ —á–µ–ª–æ–≤–µ—á–µ—Å–∫—É—é —Ä–µ—á—å –∞—É–¥–∏–æ –¥–∏–∞–ø–∞–∑–æ–Ω–∞!
    audio[k] = raw[k*2] - 127.5  # –ú–Ω–∏–º–∞—è —á–∞—Å—Ç—å q_raw –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è –≤ —É—Ç–∏–ª—å
    
# –î–∞–ª–µ–µ –∑–≤—É–∫ –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è –Ω–∞ IIR –§–í–ß –∏ –¥–µ—Ü–∏–º–∞—Ü–∏—é...
```

## 13. –ß–∞—Å—Ç–æ—Ç–Ω–∞—è –º–æ–¥—É–ª—è—Ü–∏—è (FM) –∏ –ü–æ–ª—è—Ä–Ω—ã–π –¥–∏—Å–∫—Ä–∏–º–∏–Ω–∞—Ç–æ—Ä

**–§–∏–∑–∏–∫–∞ –ø–µ—Ä–µ–¥–∞—á–∏:**
–í FM –º—ã –¥–µ—Ä–∂–∏–º –∞–º–ø–ª–∏—Ç—É–¥—É —Å—Ç—Ä–æ–≥–æ –Ω–µ–∏–∑–º–µ–Ω–Ω–æ–π (—á—Ç–æ –¥–∞–µ—Ç –∏–º–º—É–Ω–∏—Ç–µ—Ç –∫ –∞–º–ø–ª–∏—Ç—É–¥–Ω—ã–º —Ä–∞–∑—Ä—è–¥–∞–º –º–æ–ª–Ω–∏–π). –ò–Ω—Ñ–æ—Ä–º–∞—Ü–∏–æ–Ω–Ω—ã–π —Ç–æ–∫ $m(t)$ –Ω–∞–ø—Ä—è–º—É—é –Ω–∞–∫—Ä—É—á–∏–≤–∞–µ—Ç –∏–Ω—Ç–µ–≥—Ä–∞–ª –º–≥–Ω–æ–≤–µ–Ω–Ω–æ–π —Ñ–∞–∑—ã –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–∞:
`s_FM(t) = A_0 * cos(œâ_c * t  +  2*œÄ * k_f * ‚à´ m(œÑ) dœÑ)`
–ü–æ –ü—Ä–∞–≤–∏–ª—É –ö–∞—Ä—Å–æ–Ω–∞, —à–∏—Ä–∏–Ω–∞ —ç—Ñ–∏—Ä–Ω–æ–π –ø–æ–ª–æ—Å—ã FM –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è –∫–∞–∫ `–ë ‚âà 2 * (Œîf + f_m)`, –≥–¥–µ `Œîf` ‚Äî –¥–µ–≤–∏–∞—Ü–∏—è —á–∞—Å—Ç–æ—Ç—ã (—à–∏—Ä–∏–Ω–∞ —Ä–∞—Å–∫–∞—á–∫–∏ –æ—Ç –≥—Ä–æ–º–∫–æ—Å—Ç–∏), –∞ `f_m` ‚Äî –º–∞–∫—Å–∏–º–∞–ª—å–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞ –∑–≤—É–∫–∞. –í —Å—Ç–µ—Ä–µ–æ–≤–µ—â–∞–Ω–∏–∏ WBFM –¥–µ–≤–∏–∞—Ü–∏—è –¥–æ—Å—Ç–∏–≥–∞–µ—Ç `¬±75 –∫–ì—Ü`.

**–î–µ–º–æ–¥—É–ª—è—Ç–æ—Ä (–ü—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –≤–µ–∫—Ç–æ—Ä–Ω–æ–π —Ñ–∞–∑—ã):**
–ú–≥–Ω–æ–≤–µ–Ω–Ω–∞—è –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∞—è —á–∞—Å—Ç–æ—Ç–∞ ‚Äî —ç—Ç–æ –ø–µ—Ä–≤–∞—è –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è —Ñ–∞–∑—ã –ø–æ –≤—Ä–µ–º–µ–Ω–∏: `f(t) = (1 / 2*œÄ) * dŒ∏/dt`.

–¢–æ –µ—Å—Ç—å –Ω–∞–º –Ω—É–∂–Ω–æ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞—Ç—å —É–≥–æ–ª –∫–æ–º–ø–ª–µ–∫—Å–Ω–æ–≥–æ –≤–µ–∫—Ç–æ—Ä–∞ —Å–æ –≤—Ä–µ–º–µ–Ω–µ–º: 
`ŒîŒ∏_n = atan2(Q_n, I_n) - atan2(Q_n-1, I_n-1)`. 

–ù–æ –≤—ã—á–∏—Ç–∞–Ω–∏–µ —É–≥–ª–æ–≤ –ø–æ—Ä–æ–∂–¥–∞–µ—Ç –∫—Ä–∞—à —Ñ–∞–∑—ã (Phase wrapping). `(-170¬∞) - (+170¬∞) = -340¬∞`, —Ö–æ—Ç—è –≥–µ–æ–º–µ—Ç—Ä–∏—è –≥–æ–≤–æ—Ä–∏—Ç, —á—Ç–æ –∫—Ä–∞—Ç—á–∞–π—à–∏–π –ø—É—Ç—å –ø–æ –∫—Ä—É–≥—É —ç—Ç–æ `+20¬∞`.
–¢—Ä—é–∫ –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ **–ö–æ–º–ø–ª–µ–∫—Å–Ω–æ–≥–æ –°–æ–ø—Ä—è–∂–µ–Ω–∏—è**.
–í–µ–∫—Ç–æ—Ä $Z_1 = R_1 e^{j \theta_1}$, –≤–µ–∫—Ç–æ—Ä $Z_2 = R_2 e^{j \theta_2}$. –°–æ–ø—Ä—è–∂–µ–Ω–Ω—ã–π $Z_2^* = R_2 e^{-j \theta_2}$. 
–ò—Ö –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏–µ `Z_1 * Z_2^* = R_1 R_2 e^{j (\theta_1 - \theta_2)}`.
–ú—ã —Ç–æ–ª—å–∫–æ —á—Ç–æ –≤—ã—á–ª–∏ —Ñ–∞–∑—ã —á–∏—Å—Ç–æ –∞–ª–≥–µ–±—Ä–∞–∏—á–µ—Å–∫–∏–º —É–º–Ω–æ–∂–µ–Ω–∏–µ–º –º–∞—Ç—Ä–∏—Ü, –±–µ–∑ —Ç—Ä–∏–≥–æ–Ω–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–∏—Ö –≤–µ—Ç–≤–ª–µ–Ω–∏–π! –§—É–Ω–∫—Ü–∏—è `atan2` –æ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞ –Ω–∞–¥–µ–∂–Ω–æ –≤–µ—Ä–Ω–µ—Ç –Ω–∞–º —É–≥–æ–ª –æ—Ç `-œÄ` –¥–æ `+œÄ`. –≠—Ç–æ –∏ –µ—Å—Ç—å –Ω–∞—à –∏—Å—Ö–æ–¥–Ω—ã–π –∑–≤—É–∫!

```python
# –¶–∏—Ñ—Ä–æ–≤–æ–π –ø–æ–ª—è—Ä–Ω—ã–π –ß–ú –¥–∏—Å–∫—Ä–∏–º–∏–Ω–∞—Ç–æ—Ä (FM Discriminator)
prev_i, prev_q = 0.0, 0.0

for k in range(n_pairs):
    i_curr = raw[k*2] - 127.5
    q_curr = raw[k*2+1] - 127.5

    # –£–º–Ω–æ–∂–µ–Ω–∏–µ Z_curr –Ω–∞ —Å–æ–ø—Ä—è–∂–µ–Ω–Ω—ã–π Z_prev_conj
    # (i_curr + j*q_curr) * (prev_i - j*prev_q):
    p_i = i_curr * prev_i + q_curr * prev_q  # –í—ã—á–∏—Å–ª—è–µ—Ç –†–µ (Dot product / –ö–æ—Å–∏–Ω—É—Å —É–≥–ª–∞)
    p_q = q_curr * prev_i - i_curr * prev_q  # –í—ã—á–∏—Å–ª—è–µ—Ç Im (Cross product / –°–∏–Ω—É—Å —É–≥–ª–∞)

    # –ó–∞ –æ–¥–∏–Ω –≤—ã–∑–æ–≤ atan2 –º—ã –ø–æ–ª—É—á–µ–º –∏–¥–µ–∞–ª—å–Ω—É—é –æ–±–µ—Ä–Ω—É—Ç—É—é —Ä–∞–∑–Ω–∏—Ü—É (ŒîŒ∏) -> –ò—Å—Ö–æ–¥–Ω—ã–π –∑–≤—É–∫!
    audio_fm[k] = math.atan2(p_q, p_i)
    
    prev_i, prev_q = i_curr, q_curr
```

## 14. –£—Ä–∞–≤–Ω–µ–Ω–∏–µ –§—Ä–∏–∏—Å–∞ –∏ –æ–±—Ä–∞—Ç–Ω–∞—è —Å–≤—è–∑—å

–ú–æ—â–Ω–æ—Å—Ç—å —Å–∏–≥–Ω–∞–ª–∞ –∂–µ—Å—Ç–∫–æ –ø–æ–¥—á–∏–Ω—è–µ—Ç—Å—è –£—Ä–∞–≤–Ω–µ–Ω–∏—é –ø–µ—Ä–µ–¥–∞—á–∏ –§—Ä–∏–∏—Å–∞: –º–æ—â–Ω–æ—Å—Ç—å –ø–∞–¥–∞–µ—Ç –æ–±—Ä–∞—Ç–Ω–æ –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ –∫–≤–∞–¥—Ä–∞—Ç—É —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è ($P_r \sim P_t / d^2$). –î–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–π –¥–∏–∞–ø–∞–∑–æ–Ω —ç—Ñ–∏—Ä–∞ –æ–≥—Ä–æ–º–µ–Ω. –ü–µ—Ä–µ–º–Ω–æ–∂–∏—Ç–µ–ª–∏ –¶–û–° –≤—ã–¥–∞–¥—É—Ç –∑–≤—É–∫ –ª–æ–∫–∞–ª—å–Ω–æ–π —Å—Ç–∞–Ω—Ü–∏–∏ –≤–µ–ª–∏—á–∏–Ω–æ–π $100.0$, –∞ –¥–∞–ª—å–Ω–µ–π ‚Äî $0.0001$. –í—ã–≤–æ–¥ —Ç–∞–∫–∏—Ö —Ü–∏—Ñ—Ä –≤ –∑–≤—É–∫–æ–≤—É—é –∫–∞—Ä—Ç—É —á–µ—Ä–µ–∑ 16-–±–∏—Ç–Ω—ã–π PCM (`-32768..32767`) –ø—Ä–∏–≤–µ–¥–µ—Ç –∫ –∂–µ—Å—Ç–æ—á–∞–π—à–µ–º—É —Ü–∏—Ñ—Ä–æ–≤–æ–º—É –∫–ª–∏–ø–ø–∏–Ω–≥—É –ø–µ—Ä–≤–æ–≥–æ, –∏–ª–∏ –Ω—É–ª–µ–≤–æ–º—É –º–æ–ª—á–∞–Ω–∏—é –≤–æ –≤—Ç–æ—Ä–æ–º —Å–ª—É—á–∞–µ.

–ú—ã —Å—Ç—Ä–æ–∏–º **–ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫—É—é –†–µ–≥—É–ª–∏—Ä–æ–≤–∫—É –£—Å–∏–ª–µ–Ω–∏—è (–ê–†–£ / AGC)** ‚Äî –∫–æ–Ω—Ç—É—Ä –Ω–µ–ª–∏–Ω–µ–π–Ω–æ–π –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–π –æ–±—Ä–∞—Ç–Ω–æ–π —Å–≤—è–∑–∏, –∫–æ—Ç–æ—Ä—ã–π –∫–∞–∫ –Ω–µ–≤–∏–¥–∏–º—ã–π –∑–≤—É–∫–æ—Ä–µ–∂–∏—Å—Å–µ—Ä –¥–≤–∏–≥–∞–µ—Ç —Ñ–µ–π–¥–µ—Ä –≥—Ä–æ–º–∫–æ—Å—Ç–∏ `Gain`.

–ê–†–£ –ø—Ä–∏–º–µ–Ω—è–µ—Ç –¥–≤–µ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω—ã—Ö –ø–æ—Å—Ç–æ—è–Ω–Ω—ã—Ö –≤—Ä–µ–º–µ–Ω–∏ –¥–ª—è —Å–≥–ª–∞–∂–∏–≤–∞–Ω–∏—è "–æ–≥–∏–±–∞—é—â–µ–π", —á—Ç–æ–±—ã —ç–º—É–ª–∏—Ä–æ–≤–∞—Ç—å –∏–Ω–µ—Ä—Ü–∏—é —á–µ–ª–æ–≤–µ—á–µ—Å–∫–æ–≥–æ —Å–ª—É—Ö–∞:
*   **Attack (–ê—Ç–∞–∫–∞) $\tau_{att}$:** –ë—ã—Å—Ç—Ä–∞—è (–ø–æ—Ä—è–¥–∫–∞ 10‚Äì20 –º—Å). –ï—Å–ª–∏ —Å–∏–≥–Ω–∞–ª –∑–∞—à–∫–∞–ª–∏–ª, `Gain` –º–≥–Ω–æ–≤–µ–Ω–Ω–æ "–ø—Ä–æ–≤–∏—Å–∞–µ—Ç", —Å–ø–∞—Å–∞—è —É—à–∏ –æ—Ç –≤–∑—Ä—ã–≤–∞ –∫–ª–∏–ø–ø–∏–Ω–≥–∞.
*   **Release (–í–æ—Å—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏–µ) $\tau_{rel}$:** –ú–µ–¥–ª–µ–Ω–Ω–∞—è (–ø–æ—Ä—è–¥–∫–∞ 500‚Äì1000 –º—Å). –ö–æ–≥–¥–∞ –≥—Ä–æ–º–∫–∏–π –¥–∏–∫—Ç–æ—Ä –∑–∞–º–æ–ª–∫, `Gain` –≤–æ—Å—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç—Å—è –º–µ–¥–ª–µ–Ω–Ω–æ, —á—Ç–æ–±—ã —ç—Ñ–∏—Ä–Ω—ã–π —à—É–º –ø–∞—É–∑—ã –Ω–µ —É—Å–ø–µ–ª –≤–∑—Ä–µ–≤–µ—Ç—å –∫–æ—Ä–æ—Ç–∫–∏–º–∏ –Ω–∞—Å–æ—Å–Ω—ã–º–∏ —Ç–æ–ª—á–∫–∞–º–∏ ("pumping artifact").

```python
# –§–∏–∑–∏–æ–ª–æ–≥–∏—è —Ü–∏—Ñ—Ä–æ–≤–æ–≥–æ AGC
alp_att = 0.05    # –ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –±—ã—Å—Ç—Ä–æ–π –∞—Ç–∞–∫–∏ (–ó–∞—Ö–ª–æ–ø—ã–≤–∞–Ω–∏–µ –¥–∏–∞—Ñ—Ä–∞–≥–º—ã)
alp_rel = 0.0001  # –ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –º–µ–¥–ª–µ–Ω–Ω–æ–≥–æ –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏—è (–ê–¥–∞–ø—Ç–∞—Ü–∏—è –≤ —Ç–µ–º–Ω–æ—Ç–µ)
agc_v = current_state_gain

for s in float_audio:
    # –û—Ü–µ–Ω–∫–∞ –∞–º–ø–ª–∏—Ç—É–¥—ã (–í—ã—Ö–æ–¥–Ω–∞—è –º–æ—â–Ω–æ—Å—Ç—å)
    scaled = s * static_user_gain * agc_v
    env = abs(scaled)
    
    # –ù–µ–ª–∏–Ω–µ–π–Ω—ã–π –ü-—Ä–µ–≥—É–ª—è—Ç–æ—Ä –æ–±—Ä–∞—Ç–Ω–æ–π —Å–≤—è–∑–∏ (–ò–Ω—Ç–µ–≥—Ä–∞—Ü–∏—è –æ—à–∏–±–∫–∏)
    if env > 1.0:
        # –¢—è–Ω–µ–º gain –≤–Ω–∏–∑ –∫ —Ü–µ–ª–µ–≤–æ–º—É –±–µ–∑–æ–ø–∞—Å–Ω–æ–º—É (1.0 / env)
        agc_v += alp_att * (1.0 / env - agc_v)
    else:
        # –°–∏–≥–Ω–∞–ª —Ç–∏—Ö–∏–π, –ø–ª–∞–≤–Ω–æ –æ—Ç–ø—É—Å–∫–∞–µ–º gain –≤–≤–µ—Ä—Ö –∫ 1.0
        agc_v += alp_rel * (1.0 - agc_v)
        
    # –ñ–µ—Å—Ç–∫–∏–π —Ñ–∏–Ω–∞–ª–∏–∑–∏—Ä—É—é—â–∏–π clamp –¥–ª—è —É–ø–∞–∫–æ–≤–∫–∏ –≤ Int16 SoundCard
    clamped = max(-1.0, min(1.0, scaled))
    pcm_val = int(clamped * 32767.0)
```
